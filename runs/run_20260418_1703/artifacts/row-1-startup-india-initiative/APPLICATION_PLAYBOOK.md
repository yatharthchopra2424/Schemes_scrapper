# Application Playbook
## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

## Stage 1: Eligibility Verification (Day 1–3)
### Checklist
✅ Private Limited Company, LLP, or Registered Partnership
✅ Turnover < INR 100 Cr
✅ Incorporated < 10 years ago
✅ Working on innovative products/services/processes
✅ Not formed by splitting up or reconstructing an existing business

## Stage 2: Document Collection (Day 3–10)
### Universal Documents
* Certificate of Incorporation / Registration
* PAN of the entity
* Pitch deck or business description
* Authorization letter from authorized signatory
* Details of funding received (if any)

### Scheme-Specific Documents
* None required beyond universal documents

### Document Quality Check
* File size: < 5 MB
* File name: matching the document type
* Certification: self-certified
* Language: English

### Client Communication Template
"Dear [Client], we are applying for the Startup India Initiative scheme. Please find attached the required documents. The application portal is https://www.startupindia.gov.in/. We will keep you updated on the status of the application."

## Stage 3: Application Drafting (Day 10–18)
### Project Report / DPR
Not required for this scheme

### Application Form Filling Protocol
1. Register on the Startup India portal
2. Fill the DPIIT recognition form with business description
3. Upload required documents
4. Submit and await approval

### Internal Review Gate
* All fields complete: Yes
* Documents attached: Yes
* Declarations signed: Yes

## Stage 4: Submission & Follow-Up (Day 18–30)
### Submission Protocol
* Screenshot of submission: taken
* Reference number: stored
* Follow-up reminders: set

### Follow-Up Script
"Dear [Nodal Officer], this is a follow-up on our application for the Startup India Initiative scheme. The application reference number is [Reference Number]. Please let us know the status of our application."

### Escalation Path
* Level 1: Portal helpdesk (https://www.startupindia.gov.in/)
* Level 2: Nodal officer (startupindia@dipp.gov.in)
* Level 3: RTI filing if no response within statutory SLA

## Stage 5: Post-Approval (Day 30–90+)
### Approval Actions
* Approval letter: received
* Sanctioned amount: verified
* Utilization certificate: submitted
* Invoice success fee: paid
