# Application Playbook
## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

## Stage 1: Eligibility Verification (Day 1–3)
### Checklist
The following are the eligibility criteria for the Fund of Funds for Startups (FFS) scheme:
- The startup should be registered in India.
- The startup should meet the eligibility criteria set by the government.
- The startup should be working towards innovation or improvement of existing products, services, or processes and have the potential to generate employment or create wealth.
- The startup should not be formed by splitting up or reconstructing an existing business.

## Stage 2: Document Collection (Day 3–10)
### Universal Documents
The following documents are required for the FFS scheme:
- Business plan
- Financial projections
- Registration certificate
- PAN card
- Bank account details

### Scheme-Specific Documents
No additional scheme-specific documents are required for the FFS scheme.

### Document Quality Check
The following quality checks should be performed on the documents:
- File size: The file size should be within the acceptable limit.
- Name matching: The name on the documents should match the name of the startup.
- Certification requirements: The documents should be certified by the relevant authorities.
- Language: The documents should be in the required language.

## Stage 3: Application Drafting (Day 10–18)
### Project Report / DPR
A project report is not required for the FFS scheme.

### Application Form Filling Protocol
The following protocol should be followed while filling the application form:
1. Ensure that all required fields are filled.
2. Ensure that the documents are attached in the required format.
3. Ensure that the declarations are signed by the authorized signatory.

### Internal Review Gate
The following review should be performed on the application form:
- All fields are complete.
- Documents are attached in the required format.
- Declarations are signed by the authorized signatory.

## Stage 4: Submission & Follow-Up (Day 18–30)
### Submission Protocol
The application should be submitted through the Startup India portal ([https://www.startupindia.gov.in/](https://www.startupindia.gov.in/)).
- Take a screenshot of the submission confirmation.
- Store the reference number.
- Set follow-up reminders.

### Escalation Path
- Level 1: [Portal helpdesk](mailto:sidbi@sidbi.in) or helpline (1800 102 7777)
- Level 2: Nodal officer (details available on the Startup India portal)
- Level 3: RTI filing if no response within statutory SLA
