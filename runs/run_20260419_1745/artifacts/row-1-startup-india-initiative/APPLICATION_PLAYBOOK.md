# Application Playbook
## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

## Stage 1: Eligibility Verification (Day 1–3)
### Checklist
The following are the eligibility criteria for the Startup India Initiative Scheme:
- The entity should be a Private Limited Company, a Registered Partnership Firm, or a Limited Liability Partnership.
- The entity should be incorporated in India.
- The entity should have an annual turnover not exceeding INR 100 crore for any of the financial years since its incorporation.
- The entity should be working towards innovation, development, or improvement of products, services, and processes.
- The entity should have a scalable business model with a high potential for employment generation or wealth creation.
- The entity should not have been formed by splitting up or reconstructing an existing business.
- The entity should be less than 10 years old.

## Stage 2: Document Collection (Day 3–10)
### Universal Documents
The following documents are required:
- Certificate of Incorporation
- PAN of the entity
- Business description or pitch deck
- Authorization letter from the authorized signatory
- Details of funding received (if any)

### Scheme-Specific Documents
No additional scheme-specific documents are required for the Startup India Initiative Scheme.

### Document Quality Check
The following checks should be performed:
- File size: Ensure that the file size of the uploaded documents is within the allowed limit.
- Name matching: Ensure that the name of the entity matches across all documents.
- Certification requirements: Ensure that all documents are properly certified.
- Language: Ensure that all documents are in English or Hindi.

## Stage 3: Application Drafting (Day 10–18)
### Project Report / DPR
A project report or DPR is not required for the Startup India Initiative Scheme.

### Application Form Filling Protocol
The following are the critical rules for filling the application form:
- Ensure that all fields are filled correctly and completely.
- Ensure that all documents are uploaded in the correct format and size.
- Ensure that the application form is submitted within the stipulated timeframe.

### Internal Review Gate
The following is a standard review checklist:
- All fields are filled correctly and completely.
- All documents are uploaded in the correct format and size.
- The application form is submitted within the stipulated timeframe.
- The entity meets the eligibility criteria for the scheme.

## Stage 4: Submission & Follow-Up (Day 18–30)
### Submission Protocol
The following is the standard submission protocol:
- Take a screenshot of the submission confirmation page.
- Store the reference number generated after submission.
- Set follow-up reminders to track the status of the application.

### Follow-Up Script
The following is a sample follow-up script:
"Dear [Nodal Officer's Name], 
We are writing to follow up on our application for recognition under the Startup India Initiative Scheme, which was submitted on [Date of Submission]. 
The reference number for our application is [Reference Number]. 
We would appreciate it if you could provide us with an update on the status of our application.
Thank you for your time and assistance.
Best regards,
[Your Name]"

### Escalation Path
The following is the escalation path:
- Level 1: Portal helpdesk
- Level 2: Nodal officer
- Level 3: RTI filing if no response within statutory SLA

## Stage 5: Post-Approval (Day 30–90+)
### Approval Actions
The following is a standard checklist:
- Verify the approval letter.
- Verify the sanctioned amount.
- Obtain a utilization certificate.
- Invoice the client for the success fee.
