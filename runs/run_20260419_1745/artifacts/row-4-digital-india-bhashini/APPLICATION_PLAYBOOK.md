# Application Playbook
## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

## Stage 1: Eligibility Verification (Day 1–3)
### Checklist
The following checklist is based on the Digital India Bhashini Scheme's eligibility criteria:
* Registration type: All citizens, startups, entrepreneurs, and MSMEs in India are eligible.
* Turnover threshold: Not specified.
* Employee count: Not specified.
* Sector: All sectors are eligible.
* Age of business: Not specified.
* Other conditions: The programme is open to all citizens, startups, entrepreneurs, and MSMEs in India.

## Stage 2: Document Collection (Day 3–10)
### Universal Documents
The following documents are always required:
* PAN
* ITR
* GST
* Bank statements
* Udyam

### Scheme-Specific Documents
The Digital India Bhashini Scheme does not require any specific documents beyond the universal set.

### Document Quality Check
The following checklist is used to verify the quality of the documents:
* File size: Not specified.
* Name matching: The name on the documents should match the name of the applicant.
* Certification requirements: Not specified.
* Language: English or Hindi.

### Client Communication Template
The following message can be used to communicate with the client:
"Dear [Client Name], we are applying for the Digital India Bhashini Scheme on your behalf. Please find the application portal at https://digitalindia.gov.in/. The deadline for application is not specified. We will keep you updated on the status of your application."

## Stage 3: Application Drafting (Day 10–18)
### Project Report / DPR
A Project Report / DPR is not required for this scheme.

### Application Form Filling Protocol
The following rules are used to fill the application form:
1. Ensure all fields are complete and accurate.
2. Attach all required documents.
3. Sign the declarations.
4. Verify the application form for any errors.
5. Submit the application form online through the portal https://digitalindia.gov.in/.
6. Take a screenshot of the submission confirmation.
7. Store the reference number for future reference.
8. Set follow-up reminders for the application status.

### Internal Review Gate
The following checklist is used to review the application form:
* All fields are complete and accurate.
* All required documents are attached.
* Declarations are signed.
* Application form is verified for any errors.

## Stage 4: Submission & Follow-Up (Day 18–30)
### Submission Protocol
The application form is submitted online through the portal https://digitalindia.gov.in/.

### Follow-Up Script
The following script can be used to follow up with the nodal officer or portal helpdesk:
"Dear [Nodal Officer/Helpdesk], we are following up on the application for the Digital India Bhashini Scheme submitted on [Date]. The reference number is [Reference Number]. Could you please update us on the status of the application?"

### Escalation Path
The following escalation path can be used:
* Level 1: Portal helpdesk at https://digitalindia.gov.in/.
* Level 2: Nodal officer at [TO BE FILLED BY CONSULTANT].
* Level 3: RTI filing if no response within statutory SLA.

## Stage 5: Post-Approval (Day 30–90+)
### Approval Actions
The following actions are taken after approval:
* Verify the approval letter.
* Verify the sanctioned amount.
* Obtain the utilization certificate.
* Invoice the success fee.
