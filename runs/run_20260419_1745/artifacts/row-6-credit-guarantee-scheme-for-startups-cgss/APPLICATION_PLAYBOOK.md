# Application Playbook
## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

## Stage 1: Eligibility Verification (Day 1–3)
### Checklist
The following eligibility criteria must be met:
✅ The entity must be a private limited company, partnership firm, or limited liability partnership.
✅ The entity must be incorporated/registered in India.
✅ The entity must have a turnover of less than ₹100 crore in any financial year since its incorporation.
✅ The entity must be working towards innovation, development, or improvement of products or services, or have a scalable business model with high potential for employment generation or wealth creation.
✅ The entity must not be formed by splitting up or reconstructing an existing business.

## Stage 2: Document Collection (Day 3–10)
### Universal Documents
The following documents are required:
* PAN card
* ITR
* GST registration certificate
* Bank statements
* Udyam registration certificate

### Scheme-Specific Documents
The following documents are required for the GENESIS scheme:
* Certificate of incorporation
* Memorandum and Articles of Association
* Board resolution authorizing the application
* Audited financial statements for the last three years
* Business plan and project report

### Document Quality Check
The following checks must be performed:
* Verify the file size and format of the documents.
* Ensure the documents are duly signed and stamped.
* Check for any discrepancies in the information provided.

### Client Communication Template
The following message can be used to communicate with the client:
"Dear [Client], we are pleased to inform you that we will be applying for the GENESIS scheme on your behalf. Please find attached the list of required documents. We request you to provide the necessary documents at the earliest to avoid any delays in the application process. The application portal can be accessed at [insert portal URL]. The deadline for submission is [insert deadline]."

## Stage 3: Application Drafting (Day 10–18)
### Project Report / DPR
A project report is required for the GENESIS scheme. The report should include:
* Executive summary
* Business plan
* Market analysis
* Financial projections
* Management team

### Application Form Filling Protocol
The following protocol must be followed:
1. Fill out the application form online through the portal.
2. Ensure all fields are complete and accurate.
3. Upload the required documents.
4. Pay the application fee (if applicable).
5. Submit the application.

### Internal Review Gate
The following checklist must be completed:
* Verify all fields are complete and accurate.
* Ensure all required documents are attached.
* Check for any discrepancies in the information provided.

## Stage 4: Submission & Follow-Up (Day 18–30)
### Submission Protocol
The following steps must be taken:
1. Submit the application through the portal.
2. Take a screenshot of the submission confirmation.
3. Store the reference number for future reference.
4. Set follow-up reminders for the next steps.

### Follow-Up Script
The following script can be used to follow up with the nodal officer or portal helpdesk:
"Dear [Nodal Officer/Helpdesk], we are writing to follow up on our application for the GENESIS scheme. Our reference number is [insert reference number]. We would appreciate an update on the status of our application. Please let us know if any additional information is required. Thank you for your assistance."

### Escalation Path
The following escalation path can be used:
- Level 1: Portal helpdesk
- Level 2: Nodal officer
- Level 3: RTI filing if no response within statutory SLA

## Stage 5: Post-Approval (Day 30–90+)
### Approval Actions
The following actions must be taken:
* Verify the approval letter and sanctioned amount.
* Ensure the utilization certificate is submitted.
* Invoice the client for the success fee.

The application portal URL is: https://www.startupindia.gov.in/
The key sources are:
* Startup India Action Plan
* GENESIS scheme guidelines
