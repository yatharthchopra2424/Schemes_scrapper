# Application Playbook
## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

## Stage 1: Eligibility Verification (Day 1–3)
### Checklist
The following are the eligibility criteria for the GENESIS scheme:
- The entity should be a private limited company, a partnership firm, or a limited liability partnership.
- The entity should be incorporated in India.
- The entity should be less than 10 years old.
- The entity's turnover should not exceed ₹100 crore.
- The entity should be working towards innovation, development, or improvement of products or processes or services, or it should be a scalable business model with high potential for employment generation or wealth creation.

## Stage 2: Document Collection (Day 3–10)
### Universal Documents
The following documents are required:
- PAN card
- ITR
- GST registration
- Bank statements
- Udyam registration

### Scheme-Specific Documents
The following documents are required for the GENESIS scheme:
- Certificate of incorporation
- Memorandum and articles of association
- Board resolution
- Audited financial statements

### Document Quality Check
The following checks should be performed:
- File size: The file size should not exceed 5 MB.
- Name matching: The name on the documents should match the name of the entity.
- Certification requirements: The documents should be certified by a chartered accountant or a company secretary.
- Language: The documents should be in English or Hindi.

### Client Communication Template
The following message can be used to communicate with the client:
"Dear [Client], we are pleased to inform you that we will be applying for the GENESIS scheme on your behalf. Please find attached the list of documents required for the application. Kindly provide the documents at the earliest. If you have any questions or concerns, please do not hesitate to reach out to us. Thank you for choosing us as your consultant."

## Stage 3: Application Drafting (Day 10–18)
### Project Report / DPR
A project report is required for the GENESIS scheme. The report should include the following sections:
- Introduction
- Business plan
- Market analysis
- Financial projections
- Management team

### Application Form Filling Protocol
The following protocol should be followed while filling the application form:
- The form should be filled online.
- The form should be filled in English.
- The form should be filled in block letters.
- The form should be signed by the authorized signatory.

### Internal Review Gate
The following checklist should be used for internal review:
- All fields are complete
- Documents are attached
- Declarations are signed

## Stage 4: Submission & Follow-Up (Day 18–30)
### Submission Protocol
The following protocol should be followed while submitting the application:
- The application should be submitted online.
- The application should be submitted before the deadline.
- A screenshot of the submission should be taken.
- The reference number should be stored.
- Follow-up reminders should be set.

### Follow-Up Script
The following script can be used for follow-up:
"Dear [Nodal Officer], we are writing to follow up on our application for the GENESIS scheme. The application was submitted on [Date] and the reference number is [Reference Number]. We would appreciate it if you could provide us with an update on the status of our application. Thank you for your time and assistance."

### Escalation Path
The following escalation path can be used:
- Level 1: Portal helpdesk
- Level 2: Nodal officer
- Level 3: RTI filing

## Stage 5: Post-Approval (Day 30–90+)
### Approval Actions
The following actions should be taken after approval:
- Approval letter should be verified
- Sanctioned amount should be verified
- Utilization certificate should be obtained
- Invoice should be raised for success fee

The application portal URL is [https://www.startupindia.gov.in/](https://www.startupindia.gov.in/).
The key sources are:
- Startup India website
- GENESIS scheme guidelines
- Application form
- Project report template
