# Application Playbook
## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

## Stage 1: Eligibility Verification (Day 1–3)
### Checklist
✅ Registration type: Proprietorship, Partnership, Hindu Undivided Family (HUF), Cooperative, Private Limited, Public Limited
✅ Turnover threshold: Less than ₹5 crores for Micro, less than ₹50 crores for Small, less than ₹250 crores for Medium
✅ Employee count: No specific limit, but investment and turnover criteria apply
✅ Sector: Manufacturing and services
✅ Age of business: No specific limit, but registration is required
✅ Other conditions: Udyam Registration is mandatory for MSMEs to avail benefits under various schemes

## Stage 2: Document Collection (Day 3–10)
### Universal Documents
* PAN card
* Aadhaar card
* Bank account details
* GSTIN (if applicable)
* ITR (if applicable)

### Scheme-Specific Documents
* None required for Udyam Registration

### Document Quality Check
* File size: Less than 1 MB
* File name: Should match the enterprise name
* Certification requirements: Self-certification is allowed
* Language: English or regional language

### Client Communication Template
"Dear [Client], 
We are applying for Udyam Registration on your behalf. Please find the application portal at https://udyamregistration.gov.in. The deadline for registration is rolling, and there is no fixed deadline. 
Please note that Udyam Registration provides various benefits, including priority sector lending, easier access to credit, and other incentives. 
If you have any questions or concerns, please do not hesitate to reach out to us. 
Best regards, [Your Name]"

## Stage 3: Application Drafting (Day 10–18)
### Project Report / DPR
Not required for Udyam Registration

### Application Form Filling Protocol
1. Fill the application form online at https://udyamregistration.gov.in.
2. Enter the enterprise name, address, and other details accurately.
3. Upload the required documents, if any.
4. Verify the Aadhaar and PAN details.
5. Submit the application form.
6. Note the application reference number.
7. Wait for the Udyam Registration Certificate to be generated.

### Internal Review Gate
* All fields are complete
* Documents are attached (if required)
* Declarations are signed

## Stage 4: Submission & Follow-Up (Day 18–30)
### Submission Protocol
* Screenshot of submission
* Store reference number
* Set follow-up reminders

### Follow-Up Script
"Dear [Nodal Officer],
We are following up on our Udyam Registration application submitted on [Date]. The application reference number is [Reference Number]. 
Could you please update us on the status of our application? 
Thank you for your time and assistance. 
Best regards, [Your Name]"

### Escalation Path
- Level 1: Portal helpdesk (https://udyamregistration.gov.in)
- Level 2: Nodal officer (msme@nic.in, 011-23062477)
- Level 3: RTI filing if no response within statutory SLA

## Stage 5: Post-Approval (Day 30–90+)
### Approval Actions
* Verify approval letter
* Check sanctioned amount (not applicable for Udyam Registration)
* Utilization certificate (not applicable for Udyam Registration)
* Invoice success fee (if applicable)
