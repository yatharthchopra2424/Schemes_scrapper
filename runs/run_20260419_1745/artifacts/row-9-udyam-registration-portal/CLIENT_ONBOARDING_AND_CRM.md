## CRM Stage Pipeline
| Stage | Description | Action |
| --- | --- | --- |
| 01 - Lead | Initial contact made | Schedule discovery call |
| 02 - Qualified | Eligibility confirmed for Udyam Registration | Send proposal |
| 03 - Engaged | Retainer signed | Start onboarding |
| 04 - Docs collection | Gathering documents | Chase missing items |
| 05 - In preparation | Drafting application on https://udyamregistration.gov.in | Internal review |
| 06 - Submitted | Application filed on Udyam Registration Portal | Begin follow-up |
| 07 - Under review | With Ministry of Micro, Small and Medium Enterprises | Monitor + respond to queries |
| 08 - Approved | Udyam Registration Certificate received | Invoice success fee |
| 09 - Disbursed | Benefits received by client | Close case, request testimonial |
| 10 - Rejected | Application declined | Analyze, re-apply or alternate scheme |
