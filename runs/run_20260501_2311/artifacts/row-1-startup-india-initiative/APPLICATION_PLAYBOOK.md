# Application Playbook
## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

## Stage 1: Eligibility Verification (Day 1–3)
### Checklist
The following eligibility criteria must be met for the Startup India Initiative Scheme:
* The entity should be a Private Limited Company, LLP, or Registered Partnership.
* Turnover should be less than INR 100 Crores in any of the previous financial years.
* The entity should be incorporated as a Private Limited Company, a Registered Partnership Firm, or a Limited Liability Partnership.
* The entity should be working towards innovation, improvement of existing products, services, and processes, and should have the potential to generate employment or create wealth.
* The entity should not have been formed by splitting up or reconstructing an existing business.
* The entity should be less than 10 years old.

## Stage 2: Document Collection (Day 3–10)
### Universal Documents
The following documents are required:
* Certificate of Incorporation / Registration
* PAN of the entity
* Pitch deck or business description
* Authorization letter from authorized signatory
* Details of funding received (if any)

### Scheme-Specific Documents
No additional documents are specified for this scheme.

### Document Quality Check
The following quality checks should be performed on the documents:
* File size: Check that the file size is within the acceptable limit.
* Name matching: Verify that the names on the documents match the entity's name.
* Certification requirements: Ensure that the documents are certified as required.
* Language: Check that the documents are in the required language.

### Client Communication Template
The following message can be used to communicate with the client:
"Dear [Client], 
We are pleased to inform you that your application for the Startup India Initiative Scheme has been successfully submitted. 
Please find the details of the scheme and the required documents below. 
If you have any questions or concerns, please do not hesitate to reach out to us. 
Best regards, 
[Your Name]"

## Stage 3: Application Drafting (Day 10–18)
### Project Report / DPR
A project report is not required for this scheme.

### Application Form Filling Protocol
The following protocol should be followed when filling the application form:
* Ensure that all required fields are filled.
* Verify that the information provided is accurate and consistent.
* Use the correct format for dates and numbers.
* Ensure that the form is signed and dated.

### Internal Review Gate
The following review should be performed on the application form:
* Check that all required fields are filled.
* Verify that the information provided is accurate and consistent.
* Ensure that the form is signed and dated.

## Stage 4: Submission & Follow-Up (Day 18–30)
### Submission Protocol
The following protocol should be followed when submitting the application:
* Ensure that the application is submitted through the correct channel.
* Verify that the application is complete and accurate.
* Keep a record of the submission, including the date and time.

### Follow-Up Script
The following script can be used to follow up on the application:
"Dear [Authority], 
We are writing to follow up on our application for the Startup India Initiative Scheme, which was submitted on [Date]. 
Could you please provide us with an update on the status of our application? 
If there is any additional information required, please let us know. 
Thank you for your time and assistance. 
Best regards, 
[Your Name]"

### Escalation Path
The following escalation path can be used if there are any issues with the application:
* Level 1: Contact the Startup India Hub at [Phone Number] or [Email].
* Level 2: Contact the DPIIT at [Phone Number] or [Email].
* Level 3: File an RTI application if no response is received within the statutory time limit.

## Stage 5: Post-Approval (Day 30–90+)
### Approval Actions
The following actions should be taken after approval:
* Verify the approval letter and sanctioned amount.
* Ensure that the utilization certificate is submitted as required.
* Follow up on the invoice and payment process.
* Ensure that the success fee is paid as required.

Note: The application portal URL is https://www.startupindia.gov.in/. The key sources of information for this scheme are the Startup India website and the DPIIT website.
