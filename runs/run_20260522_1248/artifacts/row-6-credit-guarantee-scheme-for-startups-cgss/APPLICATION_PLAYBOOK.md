# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Borrower is a startup recognized by DPIIT as per Gazette Notifications issued from time to time  
- [ ] Borrower is not in default to any lending/investing institution  
- [ ] Borrower is not classified as a Non-Performing Asset (NPA) as per RBI guidelines  
- [ ] Borrower’s eligibility is certified by the Member Institution (MI) for the purpose of guarantee cover  
- [ ] Lending/Investing Institution is a Scheduled Commercial Bank or Financial Institution  
- [ ] Lending/Investing Institution is an RBI-registered NBFC with a rating of BBB and above by external credit rating agencies accredited by RBI  
- [ ] Lending/Investing Institution has a minimum net worth of ₹100 crore  
- [ ] Lending/Investing Institution is a SEBI-registered Alternative Investment Fund (AIF)  
- [ ] NBFCs that become ineligible due to credit rating downgrade below BBB are not eligible for further guarantee cover until upgraded again to eligible category  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Certificate of Incorporation / Registration  
- PAN Card  
- GST Registration Certificate  
- Bank Statements (last 6 months)  
- Udyam Registration Certificate  
- Income Tax Returns (ITR) for last 3 financial years  
- Audited Financial Statements for last 3 financial years  

### Scheme-Specific Documents
- DPIIT Recognition Certificate (mandatory for startup borrower)  
- Certification from Member Institution (MI) confirming borrower’s eligibility for guarantee cover  
- Signed undertaking and Board Resolution (for Member Institution registration under the scheme)  
- External credit rating report (for NBFCs as lending institution)  
- Net worth certificate (for NBFCs as lending institution)  
- SEBI registration certificate (for AIFs as lending/investing institution)  

### Document Quality Check
- All documents must be legible, scanned in PDF format  
- File size per document must not exceed 5 MB  
- Name on documents must match exactly with the applicant’s legal name  
- Certified true copies required where applicable (e.g., Board Resolution, financial statements)  
- Documents must be in English or accompanied by a certified translation  
- Financial statements must be audited and signed by a Chartered Accountant

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not explicitly required for CGSS as per evidence. However, the Member Institution (MI) must evaluate the feasibility and viability of the project per general lending protocols. **Verify with nodal officer (NCGTC) if a detailed project report is required for guarantee cover application.**

### Application Form Filling Protocol
1. Access the Jan Samarth Portal at https://www.jansamarth.gov.in/  
2. Select “Credit Guarantee Scheme for Startups (CGSS)” from the list of schemes  
3. Ensure the applicant is registered as a Member Institution (MI) on NCGTC’s portal prior to applying for guarantee cover  
4. Fill in borrower details accurately, including DPIIT recognition number  
5. Specify loan type (venture debt, working capital, subordinated debt, etc.) and amount (up to ₹20 crore per borrower)  
6. Indicate guarantee type: transaction-based or umbrella-based  
7. Attach all required documents in PDF format (max 5 MB per file)  
8. Declarations must be signed by an authorized signatory of the Member Institution  
9. Double-check that the borrower is not in default and not classified as NPA per RBI guidelines  
10. Confirm that NBFCs (if applicable) have a BBB+ rating and net worth ≥ ₹100 crore  
11. Save draft before final submission  
12. Submit only after internal review gate clearance  

### Internal Review Gate
- [ ] All applicant details (name, incorporation, PAN, GST, Udyam) are correct and complete  
- [ ] DPIIT Recognition Certificate is valid and attached  
- [ ] Member Institution eligibility confirmed (rating, net worth, SEBI registration where applicable)  
- [ ] Borrower is not in default and not classified as NPA (self-declaration or MI certification)  
- [ ] Loan amount does not exceed ₹20 crore per borrower  
- [ ] Guarantee type (transaction-based/umbrella-based) is correctly selected  
- [ ] All documents are in PDF, legible, and under 5 MB  
- [ ] Declarations are signed and dated  
- [ ] Application form is fully completed with no blank mandatory fields  
- [ ] Reference to Jan Samarth Portal and NCGTC guidelines verified

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Submit the application via the Jan Samarth Portal: https://www.jansamarth.gov.in/  
- Take a screenshot of the submission confirmation page showing the application reference number  
- Store the reference number securely in the case file  
- Set calendar reminders for follow-up at 7, 15, and 30 days post-submission  
- Save a copy of the submitted application and all attached documents

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and verify the approval letter from NCGTC via Jan Samarth Portal  
- [ ] Confirm the sanctioned guarantee amount (up to ₹20 crore per borrower)  
- [ ] Verify guarantee coverage terms:  
    - Transaction-based: 85% of amount in default for loans ≤ ₹10 crore; 75% for loans > ₹10 crore  
    - Umbrella-based: Actual losses or up to 5% of Pooled Investment, whichever is lower, capped at ₹20 crore  
- [ ] Ensure the Member Institution disburses the loan to the startup post-guarantee issuance  
- [ ] Monitor utilization of funds by the startup for intended purpose (venture debt, working capital, etc.)  
- [ ] Prepare Utilization Certificate (UC) as per NCGTC guidelines  
- [ ] Track success fee or any post-approval charges as per agreement with the client  
- [ ] Maintain records for audit and compliance purposes
