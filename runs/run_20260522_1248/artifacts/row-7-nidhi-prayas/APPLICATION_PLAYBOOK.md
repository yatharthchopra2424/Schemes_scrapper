# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is an individual innovator, startup, or student innovator from an academic institution  
- [ ] Applicant is an Indian citizen  
- [ ] Innovation is at the idea or prototype stage  
- [ ] Innovation has potential for commercialization and social impact  
- [ ] Applicant is associated with a recognized NIDHI Prayas Centre (incubator) for implementation  
- [ ] Innovation does not infringe on any existing intellectual property  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Proof of identity (Aadhaar, PAN, etc.)  
- Proof of address  
- PAN card  
- Bank statements (last 6 months)  
- Udyam Registration (if applicable)  
- GST registration (if applicable)  
- Income Tax Returns (ITR) for last 2–3 years (if applicable)  
- Certificate of Incorporation / Registration (for startups)  

### Scheme-Specific Documents
- Detailed project proposal including innovation description, objectives, and methodology  
- Team details and resumes  
- Budget estimate for prototype development  
- Any prior work or IP related to the innovation  
- Letter of support from academic institution (if applicable)  
- Declaration of originality and ownership of the idea  

### Document Quality Check
- All documents must be in PDF format  
- File size must not exceed 5 MB per document  
- File names must match the document type (e.g., "Project_Proposal.pdf", "Team_Resumes.pdf")  
- All certificates and declarations must be signed and dated  
- Documents must be in English or Hindi; if in another language, a certified translation must be attached  
- Photographs must be clear and legible

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
A detailed project proposal is required (as per evidence). It should include:  
- Innovation description  
- Objectives  
- Methodology  
- Work plan with milestones  
- Expected outcomes  
- Budget breakdown  
- Timeline for prototype development  

### Application Form Filling Protocol
1. **Identify and apply to a recognized NIDHI Prayas Centre (incubator)** — this is the first mandatory step  
2. Fill the application form with accurate details of the innovation, team, and development plan  
3. Ensure all fields are completed; leave no blank sections  
4. Attach all required documents in the specified format (PDF, <5 MB)  
5. Double-check that the declaration of originality and ownership is signed  
6. Verify that the budget estimate aligns with the proposed milestones  
7. Confirm that the innovation is at the idea or prototype stage (not commercialized product)  
8. Submit only through the NIDHI Prayas Centre — direct portal submission is not available  

### Internal Review Gate
- [ ] All application form fields completed  
- [ ] All scheme-specific documents attached and validated  
- [ ] Universal documents (ID, address, PAN, bank statements) attached  
- [ ] Declaration of originality and ownership signed and dated  
- [ ] Letter of support from academic institution attached (if applicant is a student)  
- [ ] Budget estimate is realistic and milestone-based  
- [ ] Innovation description clearly states idea/prototype stage and commercialization potential  
- [ ] Team resumes highlight relevant expertise  

---

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Submit the application package to the recognized NIDHI Prayas Centre  
- Obtain a submission receipt or acknowledgment (email or signed copy)  
- Store the reference number provided by the Centre  
- Set follow-up reminders for 7 days after submission  
- Take a screenshot of the submission confirmation for records

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Receive and review the approval letter from the NIDHI Prayas Centre  
- [ ] Verify the sanctioned amount (up to INR 10 lakhs)  
- [ ] Sign the agreement with the Centre as per milestone-based disbursement plan  
- [ ] Receive the first tranche of funds upon approval and agreement signing  
- [ ] Develop the prototype as per the agreed milestones  
- [ ] Submit progress reports and utilization certificates at each milestone  
- [ ] Receive subsequent tranches based on milestone completion and satisfactory reports  
- [ ] Submit final prototype and project report upon completion  
- [ ] Receive final payment upon successful completion and verification  
- [ ] Ensure funds are used solely for prototype development as per the approved plan  
- [ ] Maintain records of all expenditures for audit purposes  
- [ ] Explore support for IP filing and product development as offered by the Centre  
- [ ] Leverage mentorship, technical guidance, and fabrication lab access provided by the Centre  
- [ ] Network with investors and industry experts facilitated by the NIDHI Prayas Centre
