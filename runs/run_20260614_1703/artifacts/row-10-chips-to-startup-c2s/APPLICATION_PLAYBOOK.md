# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Entity type: Startup, incubator, accelerator, mentor, or investor participating in the MeitY Startup Hub ecosystem  
- [ ] Registration: Valid Certificate of Incorporation / Registration (for entities)  
- [ ] PAN: Valid PAN of the entity  
- [ ] GST: GST registration (if applicable)  
- [ ] UDYAM: UDYAM certificate (for MSMEs)  
- [ ] Sector: Operating in electronics, IT, deep-tech, or related sectors (as per program-specific focus)  
- [ ] Program-specific: Must identify and apply to a specific program under Chips to Startup (C2S) such as SAMRIDH, TIDE 2.0, GENESIS, or corporate partnerships (e.g., Micron CREST accelerator, MathWorks, Razorpay)  
- [ ] Funding status: Details of any prior funding received (if applicable)  
- [ ] Bank account: Active bank account details for fund disbursement  
- [ ] Age of business: No explicit age restriction; eligibility varies by program (e.g., early-stage, growth-stage, market-ready)  
- [ ] Employee count: No universal threshold; varies by program (e.g., TIDE 2.0 targets ICT startups; SAMRIDH focuses on scalable product-based startups)  
- [ ] Turnover threshold: No universal turnover limit; varies by program (e.g., SAMRIDH supports startups with commercial potential; GENESIS targets Tier II/III cities)  
- [ ] Geographic scope: Pan-India eligibility; no state-specific restrictions  
- [ ] Additional: Must not be blacklisted or debarred by Government of India  

> **Note**: Eligibility criteria are program-specific. Verify exact requirements for the chosen initiative (e.g., SAMRIDH, TIDE 2.0, GENESIS) on the MeitY Startup Hub portal.

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Certificate of Incorporation / Registration  
- PAN of the entity  
- Latest ITR (Income Tax Return)  
- GST registration certificate (if applicable)  
- Bank statements (last 6 months)  
- UDYAM certificate (for MSMEs)  
- Authorization letter from authorized signatory  

### Scheme-Specific Documents
- Pitch deck or business description  
- Details of funding received (if any)  
- For incubator/accelerator applicants: Proof of implementation agency status (if applying as IP)  
- For corporate partnership programs: Additional documents as specified by the partner (e.g., Micron CREST accelerator may require semiconductor domain focus)  
- For GENESIS: Proof of Tier II/III city location  
- For SAMRIDH: Evidence of scalable product-based startup with commercial potential  
- For TIDE 2.0: Focus on emerging technologies (IoT, AI, blockchain, robotics) addressing societal relevance  

### Document Quality Check
- All documents must be legible, scanned in PDF format  
- File size: Each document < 5 MB; total upload size < 25 MB  
- File naming: Use format `[DocumentType]_[EntityName]_[Date].pdf` (e.g., `CertificateOfIncorporation_XYZStartups_20250405.pdf`)  
- Name matching: Entity name must match exactly across all documents (Certificate of Incorporation, PAN, GST, UDYAM)  
- Certification: GST and UDYAM certificates must be valid and active on the date of application  
- Language: Documents in English or Hindi; if in regional language, provide certified English translation  
- Authorization letter: Must be on company letterhead, signed by authorized signatory, and specify purpose of application

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
- **DPR Requirement**: Not explicitly mandated for C2S applications. However, for funding-intensive programs like SAMRIDH (up to Rs. 40 Lacs) or GENESIS deep-tech support (up to Rs. 1 Cr), a detailed project report may be expected during evaluation.  
- **If required, standard sections should include**: Executive Summary, Problem Statement, Solution Description, Technology Innovation, Market Analysis, Business Model, Go-to-Market Strategy, Financial Projections, Funding Utilization Plan, Team Background, Risk Assessment, and Timeline.  
- **If not mentioned**: Verify with the nodal officer or scheme manager for the specific program (e.g., contact Bhuvaneshwari Nagarajan, Manager - Schemes, at manager1-msh@govcontractor.in)  

### Application Form Filling Protocol
1. **Access the correct program page**: Navigate to https://msh.meity.gov.in/ and select the specific program (e.g., SAMRIDH, TIDE 2.0) under “Programs”  
2. **Use official email**: Register using a professional domain email (e.g., info@yourstartup.com); avoid personal Gmail/Yahoo IDs  
3. **Fill all mandatory fields**: Do not leave any asterisk-marked (*) fields blank  
4. **Upload documents in prescribed format**: Only PDF/JPG/PNG; ensure file size and naming comply with quality check  
5. **Be precise in descriptions**: Use clear, concise language in pitch deck/business description; avoid jargon  
6. **Declare prior funding accurately**: Disclose all previous grants, equity, or debt funding in the “Details of funding received” field  
7. **Confirm authorization**: Ensure the authorization letter is uploaded and matches the signatory details in the form  
8. **Review before submission**: Preview the entire form; once submitted, edits are not allowed on the portal  

### Internal Review Gate
- [ ] All eligibility criteria verified and documented  
- [ ] Universal and scheme-specific documents collected and quality-checked  
- [ ] Application form filled completely; no mandatory fields left blank  
- [ ] Pitch deck/business description uploaded and reviewed for clarity  
- [ ] Authorization letter signed and matches form signatory

- [ ] GST and UDYAM (if applicable) are valid and active  
- [ ] No spelling or formatting errors in entered data  
- [ ] Final review completed by senior consultant; sign-off obtained

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Receive and verify approval letter from MeitY Startup Hub (check email and portal)  
- [ ] Confirm sanctioned amount and disbursement schedule (e.g., tranche-based release)  
- [ ] Sign and return any required agreements (e.g., grant utilization terms, equity-linked terms if applicable)  
- [ ] Utilize funds strictly as per approved budget and program guidelines  
- [ ] Maintain records of all expenditures (bills, invoices, vouchers) for audit  
- [ ] Prepare and submit Utilization Certificate (UC) within stipulated timeframe (usually within 30 days of fund utilization)  
- [ ] Coordinate with MeitY Startup Hub for benefit activation (e.g., mentorship, lab access, market connect)  
- [ ] Track and report on milestones (e.g., product development, customer acquisition, employment generated)  
- [ ] Pay success fee to consultancy firm as per agreed terms (if applicable)  
- [ ] Ensure compliance with post-approval reporting requirements (quarterly/annual progress reports)  
- [ ] Acknowledge MeitY Startup Hub support in public disclosures, press releases, and events (as per logo usage guidelines)  

> **Note**: For logo usage, prior written consent is required from MeitY Startup Hub. Refer to https://msh.meity.gov.in/about/mshlogousageguidelines
