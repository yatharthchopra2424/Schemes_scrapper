# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is an Indian startup or MSME registered under the Companies Act, 2013 or LLP Act, 2008  
- [ ] Applicant is working in the biotechnology sector  
- [ ] Applicant has an innovative idea requiring proof-of-concept or prototype development  
- [ ] Technology has potential for commercialization  
- [ ] Applicant is a legal entity registered in India  
- [ ] Preference given to ventures with strong technical feasibility and market potential  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Incorporation/registration certificate  
- PAN card  
- ITR (last 2 years)  
- GST registration certificate  
- Bank statements (last 6 months)  
- Udyam registration (for MSMEs)  

### Scheme-Specific Documents
- Project proposal detailing objectives, methodology, and expected outcomes  
- CVs of key team members  
- Proof of concept note or preliminary data (if available)  
- Bank account details  
- Authorization letter from the signatory  
- Declaration of no duplicate funding  

### Document Quality Check
- All documents must be in PDF format  
- File size must not exceed 5 MB per document  
- File names must match the document type (e.g., "Incorporation_Certificate.pdf")  
- All certificates must be clear, legible, and certified where applicable  
- Documents must be in English or Hindi; if in another language, a certified English translation must be provided  
- Signatures and dates must be visible on all relevant documents

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
A detailed project report is required as part of the application. It must include:  
- Executive summary  
- Objectives and expected outcomes  
- Methodology and work plan  
- Milestone-based timeline (project must be completed within 18 months from sanction)  
- Budget breakdown (grant up to INR 50 lakhs, disbursed in milestones)  
- Team qualifications and roles  
- Expected deliverables and proof-of-concept/prototype details  
- Commercialization potential and market analysis  
- IP strategy and technology transfer plan  
- Declaration that funds will not be used for salaries, infrastructure, or working capital  

### Application Form Filling Protocol
1. Register on the BIRAC website at https://www.birac.nic.in/  
2. Log in to the SBIRI application portal using your credentials  
3. Fill in all mandatory fields in the online application form — leave no field blank  
4. Upload documents in the exact order specified: project proposal, incorporation certificate, CVs, proof of concept, bank details, authorization letter, declaration  
5. Ensure the project title and applicant name match exactly across all documents  
6. Double-check that the declaration of no duplicate funding is signed and dated  
7. Preview the entire application before final submission  
8. Submit only after receiving an auto-generated confirmation email with a reference number  

### Internal Review Gate
- [ ] All application form fields are complete and accurate  
- [ ] All required documents are uploaded and clearly legible  
- [ ] File names follow the naming convention (e.g., "Project_Proposal_SBIRI.pdf")  
- [ ] Declaration of no duplicate funding is signed and dated  
- [ ] Authorization letter is on letterhead and signed by an authorized signatory  
- [ ] Project proposal includes milestone-based budget and 18-month timeline  
- [ ] No mention of fund usage for salaries, infrastructure, or working capital  
- [ ] Application is submitted via https://www.birac.nic.in/  
- [ ] Screenshot of submission confirmation is saved

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take a screenshot of the submission confirmation page showing the application reference number  
- Store the reference number in the case file and email it to the client  
- Save a PDF copy of the submitted application for records  
- Set a follow-up reminder for 15 days after submission to check status  
- Mark calendar for periodic review cycles (as per rolling basis)

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Receive and verify the approval letter from BIRAC specifying the sanctioned amount (up to INR 50 lakhs)  
- [ ] Confirm the milestone-based disbursement schedule in the agreement  
- [ ] Sign the grant agreement with BIRAC within the stipulated time  
- [ ] Utilize funds strictly for proof-of-concept and prototype development (not for salaries, infrastructure, or working capital)  
- [ ] Submit periodic progress reports and utilization certificates as per the agreement  
- [ ] Disclose any intellectual property generated to DBT/BIRAC  
- [ ] Ensure project completion within 18 months from date of sanction  
- [ ] Prepare and submit final utilization certificate and project completion report  
- [ ] Coordinate with BIRAC for technology transfer and scaling support if applicable  
- [ ] Track and report invoice for success fee as per consultancy agreement
