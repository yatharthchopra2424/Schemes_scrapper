# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is a State Government or Union Territory Administration  
- [ ] Proposal involves Digital Public Infrastructure (DPI) for Agriculture  
- [ ] Proposal involves Digital General Crop Estimation Survey (DGCES)  
- [ ] Proposal involves support to Mahalanobis National Crop Forecasting Centre (MNCFC)  
- [ ] Proposal involves IT initiatives for States, Academic & Research Institutions under the Mission  
- [ ] Proposal includes a Detailed Project Report (DPR)  
- [ ] Proposal includes financial implication details  
- [ ] Proposal includes clear timelines  
- [ ] Proposal includes technology involved  
- [ ] Proposal includes cost-benefit analysis (for feedback and monitoring)  
- [ ] Proposal is submitted preferably in the first quarter but not later than the second quarter of the Financial Year  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Certificate of Incorporation / Registration  
- PAN Card  
- Audited Financial Statements / ITR (last 3 years)  
- GST Registration Certificate  
- Bank Statements (last 6 months)  
- Udyam Registration (if applicable)  
- Authorization Letter / Board Resolution  

### Scheme-Specific Documents
- Detailed Project Report (DPR)  
- Financial implication details  
- Clear timelines  
- Technology involved  
- Cost-benefit analysis  

### Document Quality Check
- File size: Each document < 10 MB  
- File name: Must match the document type (e.g., DPR_StateName.pdf)  
- All documents must be legible and in English or Hindi  
- Scanned copies must be certified true copies where applicable  
- DPR must be signed and stamped by the competent authority

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Yes, a Detailed Project Report (DPR) is required. Standard sections include:  
- Executive Summary  
- Project Objectives  
- Scope of Work  
- Technology Involved (AI, ML, Blockchain, IoT, Robotics, etc.)  
- Implementation Plan with Clear Timelines  
- Financial Implications and Budget Estimate  
- Expected Outcomes and Benefits  
- Cost-Benefit Analysis  
- Sustainability and Scalability Plan  
- Annexures (supporting documents, letters of support, etc.)  

### Application Form Filling Protocol
1. Access the application portal at https://agriwelfare.gov.in/en/DigiAgriDiv  
2. Ensure all mandatory fields are filled; incomplete forms will be rejected  
3. Upload documents in PDF format only, with file names matching the document type  
4. Double-check that the DPR includes all required components: financials, timelines, technology, and cost-benefit analysis  
5. Confirm that the proposal aligns with the mission’s focus on Digital Public Infrastructure (DPI), DGCES, MNCFC support, or IT initiatives for States/Academic Institutions  
6. Verify that the submission timeline adheres to the preference for Q1 and latest by Q2 of the Financial Year  
7. Save a copy of the submitted application and note the application reference number  

### Internal Review Gate
- [ ] All eligibility criteria verified and documented  
- [ ] All universal and scheme-specific documents collected and quality-checked  
- [ ] DPR is complete with all required sections  
- [ ] Application form is fully filled with accurate information  
- [ ] All documents are attached and correctly named  
- [ ] Declarations and undertakings are signed where required  
- [ ] Submission is planned within the preferred timeline (Q1, latest Q2 of FY)

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take a screenshot of the submission confirmation page showing the reference number and timestamp  
- Store the application reference number securely in the case file  
- Set calendar reminders for follow-up at 15 days and 30 days post-submission  
- Monitor the PFMS portal for status updates on fund release tranches

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and archive the official approval letter from the Department of Agriculture & Farmers Welfare  
- [ ] Verify the sanctioned amount against the proposed budget  
- [ ] Prepare and submit the first utilization certificate as per PFMS requirements  
- [ ] Set up tracking for the 4 equal tranches of fund release  
- [ ] Ensure submission of satisfactory status reports via PFMS before requesting each subsequent installment  
- [ ] Prepare and share cost-benefit analysis with the Central Government as part of feedback project monitoring  
- [ ] Monitor project timelines; note that extension up to one year at a time is allowed with prior approval, up to a maximum of two years  
- [ ] Prepare for surrender of unspent amount if project is not completed within the extended period  
- [ ] Invoice success fee as per consultancy agreement terms
