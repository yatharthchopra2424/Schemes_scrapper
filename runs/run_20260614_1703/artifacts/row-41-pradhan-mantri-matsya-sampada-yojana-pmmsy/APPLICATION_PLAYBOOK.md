# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Beneficiary is a fisher, fish farmer, fish vendor, fish worker, SC/ST, woman, entrepreneur, startup, self-help group, cooperative, NGO, private company, state government, union territory, or other entity engaged in fisheries and aquaculture activities  
- [ ] Project activity is eligible under PMMSY (e.g., construction of ponds, cages, hatcheries, biofloc units, ice plants, cold storage, fish kiosks, value addition units, aquaculture in saline/alkaline lands, seaweed cultivation, ornamental fisheries, integrated aquaparks, nucleus breeding centres, fishing vessel upgradation, bio-toilets, safety kits, livelihood and nutritional support during fishing bans, training and skill development, technology adoption, market linkage, branding, export promotion)  
- [ ] Project complies with state fisheries acts, environmental norms, and Coastal Regulation Zone (CRZ) guidelines  
- [ ] Beneficiary bears the non-subsidiary portion of the project cost  
- [ ] Beneficiary does not avail similar subsidy for the same activity from other central or state schemes  
- [ ] Detailed Project Report (DPR) prepared as per guidelines  
- [ ] Land ownership/lease documents available  
- [ ] Bank account details of beneficiary ready  
- [ ] PAN/Aadhaar of beneficiary available  
- [ ] Quotations for equipment and civil works obtained  
- [ ] No Objection Certificate (NOC) from concerned departments (if applicable)  
- [ ] Environmental clearance (if required)  
- [ ] Proof of beneficiary contribution  
- [ ] Photographs of site before intervention available

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Incorporation certificate / Partnership deed / Proprietorship declaration  
- PAN card of beneficiary/entity  
- Aadhaar card of beneficiary  
- ITR of last 2–3 years (if applicable)  
- GST registration certificate  
- Bank statements (last 6 months)  
- Udyam Registration (for MSMEs)  
- Cancelled cheque / Bank passbook copy  

### Scheme-Specific Documents
- Detailed Project Report (DPR)  
- Land ownership/lease documents  
- Bank account details  
- PAN/Aadhaar of beneficiary  
- Quotations for equipment and civil works  
- No Objection Certificate (NOC) from concerned departments (if applicable)  
- Environmental clearance (if required)  
- Beneficiary contribution proof  
- Photographs of site before and after intervention  

### Document Quality Check
- All documents must be legible, scanned in PDF format  
- File size per document: < 5 MB  
- File names must follow format: `SchemeName_DocumentType_BeneficiaryName_Date.pdf`  
- All certificates must be self-attested or gazetted officer-attested  
- Photographs must be timestamped and show clear site conditions  
- Bank account details must match beneficiary name and be active  
- DPR must be signed and stamped by beneficiary and preparing agency

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
**Yes, DPR is required.**  
Standard sections to include:  
- Executive Summary  
- Project Background and Justification  
- Objectives and Outcomes  
- Technical Details (technology, species, scale)  
- Site Details (location, land/water area, maps)  
- Implementation Schedule (Bar Chart/Gantt)  
- Cost Estimate (item-wise, with quotations)  
- Funding Pattern (subsidy, beneficiary contribution, loans)  
- Economic Analysis (IRR, BCR, payback period)  
- Social Impact (employment generation, women/SC/ST inclusion)  
- Environmental and CRZ Compliance  
- Risk Assessment and Mitigation Plan  
- Monitoring and Evaluation Framework  
- Annexures: Land documents, quotations, NOCs, photographs, beneficiary contribution proof  

### Application Form Filling Protocol
1. Access the application via District Fisheries Officer (DFO) – **not directly online**; DFO forwards proposal to State Fisheries Department  
2. Ensure all fields in DPR are complete and match supporting documents  
3. Use official PMMSY guidelines for cost norms and unit rates  
4. Clearly state beneficiary category (general/SCST/women/NER/Himalayan) for correct subsidy rate (40–60% or 50–70%)  
5. Declare that no similar central/state subsidy is availed for the same activity  
6. Attach all mandatory documents as per checklist  
7. Sign and stamp DPR; beneficiary and preparing agency must authenticate  
8. Submit to District Fisheries Officer (DFO) of respective district  

### Internal Review Gate
- [ ] All eligibility criteria verified and documented  
- [ ] DPR complete with all sections and annexures  
- [ ] All scheme-specific and universal documents attached  
- [ ] Beneficiary contribution proof clearly shown  
- [ ] Quotations match cost estimates in DPR  
- [ ] Land documents clear and dispute-free  
- [ ] Bank account details validated  
- [ ] Declarations signed (no duplicate subsidy, compliance with norms)  
- [ ] Photographs before intervention submitted  
- [ ] Application routed to District Fisheries Officer (DFO) for forwarding

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Obtain receipt/acknowledgment from District Fisheries Officer (DFO) upon submission  
- Photograph the submitted DPR package and receipt  
- Store reference number/Diary number provided by DFO  
- Set follow-up reminders:  
  - Day 25: Check with DFO on forwarding to State Fisheries Department  
  - Day 35: Follow up on State Level Sanctioning Committee (SLSC) review  
  - Day 45: Enquire about submission to Department of Fisheries, GoI

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and verify approval letter from State Implementing Agency / Department of Fisheries, GoI  
- [ ] Confirm sanctioned amount and subsidy percentage (40–60% general, 50–70% for SC/ST/women/NER/Himalayan)  
- [ ] Ensure beneficiary arranges non-subsidiary portion before fund release  
- [ ] Monitor installment release based on physical and financial progress  
- [ ] Collect and submit Utilization Certificates (UC) and completion reports after each installment  
- [ ] Ensure project completion as per DPR timelines  
- [ ] Obtain final completion report and site photographs after intervention  
- [ ] Verify that no similar central/state subsidy was claimed for the same activity  
- [ ] Prepare and submit invoice for success fee as per consultancy agreement  
- [ ] Guide beneficiary on post-approval compliance (e.g., maintenance, reporting)
