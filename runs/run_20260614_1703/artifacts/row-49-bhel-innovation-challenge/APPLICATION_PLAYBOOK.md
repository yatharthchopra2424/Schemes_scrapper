# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is a startup, MSME, individual innovator, or entity registered in India
- [ ] Innovation is relevant to BHEL's operational domains (power generation, transmission, transportation, defence, aerospace, renewable energy, or industrial systems)
- [ ] Solution is at a stage suitable for demonstration, pilot testing, or potential integration with BHEL's products or services
- [ ] No explicit turnover, employee count, or age-of-business thresholds specified in evidence

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Incorporation/Registration Certificate
- PAN Card
- GST Registration (if applicable)
- Udyam Registration (for MSMEs)
- Bank Statements (last 6 months)
- Income Tax Returns (last 2 years)

### Scheme-Specific Documents
- Proposal document detailing the innovation
- Proof of entity registration (for startups/MSMEs)
- Brief profile of the applicant or team
- Technical specifications or schematics of the solution
- Any existing prototypes, test results, or pilot data
- Declaration of originality and intellectual property rights

### Document Quality Check
- All documents must be legible scans or certified copies
- File names must match the document type and applicant name
- PDF format preferred; file size under 10 MB per document
- Declarations must be signed and dated
- Technical documents must include clear descriptions and diagrams where applicable

### Client Communication Template
**Subject:** Application Invitation: BHEL Innovation Challenge – Submit Your Innovation Proposal  

Dear [Client Name],  

We invite you to apply for the **BHEL Innovation Challenge**, a recognition-based scheme by Bharat Heavy Electricals Limited (BHEL) aimed at identifying innovative solutions in power generation, transmission, transportation, defence, aerospace, renewable energy, or industrial systems.  

This is a rolling basis opportunity with no fixed deadline. Selected participants gain recognition, access to BHEL’s expertise and infrastructure, mentorship, technical guidance, and potential pathways for pilot projects, proof-of-concept development, and commercialization.  

**Key Requirements:**  
- Submit a detailed proposal outlining your innovation, its relevance, feasibility, and potential impact  
- Include proof of registration, team profile, technical specs, prototype/data (if available), and IP declaration  
- Applications are submitted via the BHEL website: https://www.bhel.com  

Please let us know if you wish to proceed, and we will guide you through the eligibility check and document preparation.

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not explicitly required per evidence. However, a detailed proposal document is mandatory. If a formal DPR is expected, verify with the nodal officer or BHEL’s innovation desk.

### Application Form Filling Protocol
1. Access the application portal via https://www.bhel.com (check for dedicated innovation challenge section)  
2. Review active problem statements or challenge themes released by BHEL  
3. Prepare a proposal covering: innovation description, relevance to BHEL domains, feasibility, impact, and implementation plan  
4. Attach all scheme-specific documents in prescribed format  
5. Ensure declaration of originality and IP rights is signed  
6. Double-check that all files are named clearly (e.g., Proposal_[ApplicantName].pdf)  
7. Submit only through the specified online or offline channel  
8. Save confirmation screenshot and application reference number  

### Internal Review Gate
- [ ] All eligibility criteria confirmed  
- [ ] Proposal document is complete and technically sound  
- [ ] All scheme-specific documents are attached and legible  
- [ ] Declarations are signed and dated  
- [ ] Application form fields are fully filled  
- [ ] Copy of submission saved for records  

---

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Submit application via the mode specified (online/offline) on https://www.bhel.com  
- Take screenshot of submission confirmation page  
- Save application reference number or acknowledgment ID  
- Log submission date and set follow-up reminder for 15–20 days

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and review approval/intimation letter from BHEL  
- [ ] Confirm sanctioned support (mentorship, facility access, collaboration terms)  
- [ ] Prepare utilization plan for availed benefits (e.g., lab access, mentorship hours)  
- [ ] Track progress against agreed milestones (if any)  
- [ ] Maintain records for potential invoice or success fee (if post-selection agreement applies)  
- [ ] Ensure compliance with IP terms mutually agreed with BHEL
