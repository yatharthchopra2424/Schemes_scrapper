# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Startup is registered under the Companies Act, 2013, LLP Act, 2008, or as a partnership firm  
- [ ] Working on innovative products, services, or processes in domains such as renewable energy, green hydrogen, biofuels, carbon capture, AI/ML for energy optimization, or sustainable mobility  
- [ ] Has a working prototype or minimum viable product (MVP) (preferred)  
- [ ] Startup is not more than 10 years old from the date of incorporation  
- [ ] Application submitted via the designated online portal or email address as specified on the IOCL website  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Certificate of Incorporation / Registration  
- PAN of the entity  
- Audited financial statements (if available)  
- GST registration certificate  
- Bank statements (last 6 months)  
- Udyam Registration (if applicable)  

### Scheme-Specific Documents
- Detailed project proposal (including problem statement, solution, technology readiness, team details, and potential impact)  
- Proof of concept or prototype details (if any)  
- Team resumes and background  
- Intellectual property details (if applicable)  
- Declaration of compliance with eligibility criteria  

### Document Quality Check
- All documents must be in PDF or image format (JPEG/PNG)  
- File size must not exceed 5 MB per document  
- File names must match the document type (e.g., "Certificate_of_Incorporation.pdf")  
- All certificates must be clear, legible, and issued by competent authorities  
- Documents must be in English or accompanied by a certified translation  
- Declaration must be signed and dated by the authorized signatory

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
A detailed project proposal is required (as per evidence). It should include:  
- Problem statement  
- Proposed solution  
- Technology readiness level  
- Team details and background  
- Potential impact and alignment with IOCL’s strategic goals (renewable energy, green hydrogen, biofuels, carbon capture, AI/ML for energy optimization, sustainable mobility)  
- Commercialization or scalability potential  

### Application Form Filling Protocol
1. Visit the official IOCL website ([https://iocl.com](https://iocl.com)) and navigate to the Startup Scheme section.  
2. Review eligibility criteria and focus areas (renewable energy, green hydrogen, biofuels, carbon capture, AI/ML for energy optimization, sustainable mobility).  
3. Prepare a detailed proposal as per the structure above.  
4. Ensure all fields in the application form are completed accurately.  
5. Attach all required documents in the specified format (PDF/JPEG, <5 MB each).  
6. Verify that the declaration of compliance is signed and dated.  
7. Submit via the designated online portal or email address as specified.  
8. Save a screenshot of the submission confirmation and note the reference number.  

### Internal Review Gate
- [ ] All application form fields completed  
- [ ] All scheme-specific documents attached  
- [ ] Universal documents (PAN, Incorporation, GST, bank statements, Udyam) attached  
- [ ] Declaration of compliance signed and dated  
- [ ] File sizes within limits (<5 MB per document)  
- [ ] File names are clear and match document type  
- [ ] Proposal aligns with IOCL’s strategic objectives (energy, sustainability, innovation)  
- [ ] Submission screenshot and reference number saved

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take a screenshot of the submission confirmation page showing the reference number and timestamp  
- Store the reference number securely in the case file  
- Set a follow-up reminder for 15 days post-submission  
- Save all submitted documents in a dedicated folder labeled with Case ID and scheme name

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and review the approval letter or MoU/project agreement from IOCL  
- [ ] Verify the sanctioned amount and disbursement schedule (grant, seed investment, or milestone-based)  
- [ ] Prepare utilization certificate as per IOCL’s requirements  
- [ ] Ensure compliance with IOCL’s safety, security, and confidentiality protocols  
- [ ] Track milestone achievement for subsequent disbursements  
- [ ] Submit progress reports as agreed in the MoU/project agreement  
- [ ] Coordinate with IOCL’s R&D and business teams for mentorship and technical support  
- [ ] Explore opportunities for proof-of-concept development, joint projects, and potential commercialization or procurement  
- [ ] Plan for scaling solutions with IOCL’s market reach and infrastructure access  
- [ ] Invoice success fee as per consultancy agreement (to be filled by consultant)
