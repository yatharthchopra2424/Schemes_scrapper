# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is an individual seeking skill development training  
- [ ] Applicant is an Indian citizen  
- [ ] Applicant is aged between 15–59 years (as per standard Skill India guidelines)  
- [ ] Applicant has access to a smartphone or computer with internet connectivity  
- [ ] Applicant is not currently enrolled in a full-time regular academic program (unless seeking upskilling/reskilling)  
- [ ] Applicant belongs to a priority group (if applicable): SC/ST, OBC, women, persons with disabilities, or economically weaker sections  
- [ ] Applicant has a valid Aadhaar number for identity verification  
- [ ] Applicant is registering for a course listed on the Skill India Digital Hub (SIDH) portal  

> **Note**: The Skill India Digital scheme (via SIDH) primarily targets individual learners for skill training. No turnover, employee count, or business registration criteria apply. Eligibility is based on individual learner profile.  
> **Source**: https://skillindia.gov.in/ (SIDH Admin Portal, MSDE)

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- [ ] Aadhaar Card (for identity and KYC verification)  
- [ ] PAN Card (for financial tracking and certification linkage)  
- [ ] Bank Account Details (including IFSC code and account number for stipend or reimbursement, if applicable)  
- [ ] Passport-sized photograph (recent, white background)  
- [ ] Mobile number and email ID (for OTP verification and communication)  

### Scheme-Specific Documents
- [ ] Educational qualification certificates (if applying for advanced or specialized courses)  
- [ ] Caste certificate (if claiming benefits under SC/ST/OBC categories)  
- [ ] Disability certificate (issued by authorized medical authority, if applying under PwD quota)  
- [ ] Income certificate (if applying under economically weaker sections or for fee waiver)  
- [ ] Domicile certificate (if state-specific course or reservation applies)  
- [ ] Current employment or unemployment proof (if applicable for wage-linked or placement-linked training)  

> **Note**: Document requirements vary by course and implementing partner. Universal documents (Aadhaar, PAN, bank) are mandatory for portal registration. Additional documents are course- or beneficiary-specific.  
> **Source**: https://skillindia.gov.in/ (SIDH portal implies standard KYC and beneficiary profiling)

### Document Quality Check
- [ ] All documents are clear, legible scans or high-quality photographs  
- [ ] File size does not exceed 2 MB per document  
- [ ] File names follow format: `CaseID_DocumentType_Name.pdf` (e.g., `CASE001_Aadhaar_RamSharma.pdf`)  
- [ ] Aadhaar number is masked (showing only last 4 digits) in uploaded copies for data privacy  
- [ ] Photograph is recent (within last 6 months), with plain background and face clearly visible  
- [ ] Certificates are self-attested if not gazetted; gazetted copies preferred for official categories  
- [ ] All documents are in English or Hindi; if in regional language, include certified translation  
- [ ] Bank details include account holder name matching Aadhaar/PAN, IFSC code, and account number

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
- **Not required**. The Skill India Digital scheme (SIDH) is an individual beneficiary-facing platform for skill training enrollment. No Detailed Project Report (DPR) or business proposal is needed.  
- **Action**: Verify course-specific prerequisites (if any) with the course provider or nodal officer before enrollment.  

### Application Form Filling Protocol
1. Access the official portal: **https://skillindia.gov.in/** (Skill India Digital Hub - SIDH)  
2. Click on “Login” or “Register” to begin the application process  
3. Use Aadhaar number for OTP-based authentication (mandatory for identity verification)  
4. Fill in personal details exactly as per Aadhaar (name, DOB, gender, address)  
5. Provide valid mobile number and email ID for OTP and communication  
6. Select preferred skill course(s) from the catalog based on interest, qualification, and career goals  
7. Upload all required documents in the specified format and size  
8. Review all entries before final submission; incorrect Aadhaar or bank details may lead to rejection  
9. Submit the form and save the application reference number  
10. Do not use third-party intermediaries for registration; direct portal use is encouraged  

> **Portal Note**: The SIDH portal is administered by MSDE and may redirect to partner platforms for specific courses. Always verify authenticity of links.  
> **Source**: https://skillindia.gov.in/ (SIDH Admin Portal, MSDE)

### Internal Review Gate
- [ ] Aadhaar number entered correctly and matches uploaded document  
- [ ] PAN details provided and verified (if applicable for stipend linkage)  
- [ ] Bank account details accurate and match beneficiary name  
- [ ] Contact details (mobile, email) are active and accessible  
- [ ] All mandatory documents uploaded and legible  
- [ ] Course selection aligns with client’s qualification and career objective  
- [ ] Declarations regarding data privacy and consent are accepted  
- [ ] Application is submitted directly via https://skillindia.gov.in/ (no third-party links)  
- [ ] Reference number generated and recorded in case file

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take a screenshot of the submission confirmation page showing the **Application Reference Number**  
- Save the reference number in the case file and communicate it to the client  
- Download and store a copy of the submitted application form (if available)  
- Set a follow-up reminder for **7 days after submission** to check enrollment status  
- Monitor the registered email and SMS for OTPs, course allocation updates, or login credentials

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Confirm course enrollment and access to learning materials on SIDH or partner platform  
- [ ] Verify login credentials and course validity period  
- [ ] Track completion milestones and assessment schedules  
- [ ] Ensure client receives certificate upon successful course completion (issued by NCVET/SCVT or partner institute)  
- [ ] Monitor for any stipend, placement support, or wage linkage (if applicable under specific course)  
- [ ] Collect feedback on training quality and placement outcome  
- [ ] Assist in uploading utilization certificate or completion proof if required for future schemes  
- [ ] Invoice success fee only after verified course completion or placement (as per firm policy)  

> **Note**: Post-approval actions focus on learning outcomes, certification, and employability. No financial disbursement to consultants is involved unless tied to placement-linked programs.  
> **Source**: https://skillindia.gov.in/ (SIDH implies training, certification, and placement support)
