# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is a registered entity under Indian law (Company, LLP, Partnership Firm, Proprietorship, Society, Trust, or Cooperative Society)
- [ ] Entity is engaged in manufacturing, services, agriculture, or allied activities related to export promotion
- [ ] Annual turnover does not exceed ₹500 crore (as per latest audited financial statements)
- [ ] Entity has been in operation for at least 3 years as of the date of application
- [ ] Project involves creation or upgradation of trade infrastructure directly linked to export promotion (e.g., testing labs, warehouses, cold storage, cargo handling, ICT infrastructure)
- [ ] Project is located in India and intended to benefit exporters (including MSMEs) in the region
- [ ] Applicant has not previously received funding for the same project under any other central government scheme
- [ ] Applicant possesses clear title or lease rights to the land where infrastructure is to be developed (minimum lease period of 10 years)
- [ ] Project proposal includes a detailed feasibility study and cost-benefit analysis
- [ ] Applicant agrees to bear at least 25% of the project cost as promoter’s contribution (remaining up to 75% may be funded under TIES)

> **Note**: Eligibility criteria are inferred from scheme objectives and typical TIES guidelines, as the official portal (https://commerce.gov.in/ties) returned a 404 error. Verify current criteria with the Department of Commerce, Ministry of Commerce & Industry.

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- [ ] Incorporation/Registration Certificate (Company/LLP/Firm/etc.)
- [ ] PAN Card of the entity
- [ ] Audited Financial Statements (ITR) for last 3 years
- [ ] GST Registration Certificate
- [ ] Bank Statements (last 6 months)
- [ ] Udyam Registration Certificate (if applicable for MSMEs)
- [ ] Cancelled Cheque / Bank Account Verification Letter
- [ ] Authorization Letter / Board Resolution for application submission

### Scheme-Specific Documents
- [ ] Detailed Project Report (DPR) including technical feasibility, cost estimates, and implementation timeline
- [ ] Land documents: Sale deed, lease agreement (minimum 10 years), or allotment letter from government agency
- [ ] No Objection Certificate (NOC) from local authorities (pollution, fire, municipal, etc.)
- [ ] Clearance from relevant sectoral regulator (if applicable, e.g., FSSAI for food processing warehouses)
- [ ] Export performance proof: IEC certificate, RCMC, and export invoices/bills of lading for last 2 years
- [ ] List of intended beneficiary exporters (with IEC details) and their projected usage of the infrastructure
- [ ] Cost-benefit analysis and social impact assessment report
- [ ] Environmental clearance certificate (if project attracts EPA 1986 provisions)
- [ ] Architectural/engineering drawings and structural safety certificate
- [ ] Undertaking that promoter’s contribution (minimum 25%) is arranged and unencumbered
- [ ] Undertaking that no duplicate funding has been sought/received for same project under other GoI schemes

### Document Quality Check
- [ ] All documents are legible, scanned in PDF format (minimum 300 DPI)
- [ ] File names follow format: `SchemeName_DocumentType_EntityName_Date.pdf` (e.g., `TIES_LandLease_ABCExports_20240515.pdf`)
- [ ] All certificates are valid and not expired (check issue/expiry dates)
- [ ] Financial documents are audited and signed by CA with UDIN
- [ ] Land documents are registered and duly stamped
- [ ] Declarations and undertakings are on non-judicial stamp paper of appropriate value (as per state stamp act)
- [ ] Documents are in English or Hindi; if in regional language, attach certified translation
- [ ] Total upload size does not exceed portal limit (verify with nodal officer; typically 25–50 MB per file)

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
- **DPR Required**: Yes — a Detailed Project Report is mandatory for TIES applications.  
- **Standard Sections to Include**:  
  1. Executive Summary  
  2. Project Promoters’ Profile and Credentials  
  3. Project Rationale and Linkage to Export Promotion  
  4. Technical Description (layout, capacity, technology, utilities)  
  5. Site Details (location, land status, connectivity)  
  6. Implementation Schedule (Gantt chart)  
  7. Cost Estimates (item-wise, with quotations)  
  8. Means of Finance (promoter’s contribution vs. TIES grant)  
  9. Working Capital Requirements  
  10. Projected Income and Expenditure Statement (5 years)  
  11. Financial Analysis (IRR, NPV, payback period)  
  12. Social Cost-Benefit Analysis  
  13. Environmental and Safety Measures  
  14. Export Impact Assessment (number of beneficiaries, projected export increase)  
  15. Risk Analysis and Mitigation Plan  
  16. Annexures: Land documents, quotations, approvals, promoter’s financials  

> **Note**: Confirm DPR format expectations with the nodal officer, as guidelines may vary.

### Application Form Filling Protocol
1. Access the official TIES application portal (verify active URL with nodal officer; current link https://commerce.gov.in/ties returns 404).  
2. Use only the latest version of the application form; do not use outdated templates.  
3. Fill all fields in English; avoid abbreviations unless explicitly allowed.  
4. Ensure numeric fields (costs, turnover, etc.) use Indian numbering format (lakhs/crores) without commas or symbols (e.g., 50000000 for ₹5 crore).  
5. Match entity name exactly as per PAN and incorporation certificate across all fields.  
6. Enter promoter’s contribution amount and confirm it meets ≥25% of total project cost.  
7. Upload documents in the specified order and format as per portal instructions; validate each upload before proceeding.  
8. Preview the entire form before final submission; once submitted, edits may not be allowed.  
9. Save a PDF copy of the submitted application and retain the application reference number.  
10. Do not submit if any mandatory field is blank or document is missing — use “Not Applicable” only if explicitly permitted in the form.

### Internal Review Gate
- [ ] All personal and entity details match PAN, incorporation certificate, and bank records  
- [ ] Project cost breakdown is realistic and supported by quotations  
- [ ] Promoter’s contribution is ≥25% and verifiable from bank statements  
- [ ] All universal and scheme-specific documents are attached and correctly named  
- [ ] Declarations (no duplicate funding, land title, promoter’s commitment) are signed and dated  
- [ ] Authorized signatory is verified via board resolution or authorization letter  
- [ ] Application form has no blank mandatory fields  
- [ ] Digital signatures (if required) are applied using valid DSC  
- [ ] Application reference number is generated and recorded upon submission  
- [ ] Screenshot of confirmation page is saved with timestamp

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and verify the official approval letter from the Department of Commerce specifying sanctioned TIES grant amount  
- [ ] Confirm that the sanctioned amount aligns with the eligible project cost (max 75% of total, subject to ceiling)  
- [ ] Register the project on the Public Financial Management System (PFMS) if required for fund tracking  
- [ ] Utilize funds strictly as per approved DPR and submit utilization certificates (UCs) in GFR 12-A format  
- [ ] Ensure timely completion of project milestones as per implementation schedule  
- [ ] Maintain separate bank account for TIES funds (if mandated)  
- [ ] Notify the nodal officer of project completion and request for final inspection  
- [ ] Submit final utilization certificate, audit report, and project completion report within stipulated time  
- [ ] Monitor and report on actual export benefits achieved (e.g., number of exporters using facility, export growth)  
- [ ] Coordinate with client to calculate and collect success fee as per engagement agreement  
- [ ] Archive all approvals, UCs, correspondence, and completion documents in the case file  

> **Note**: Post-approval compliance is critical; failure to submit UCs or divert funds may trigger recovery with interest. Confirm exact reporting frequency and format with the nodal officer.
