# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Organisation is registered as an NGO/Trust/Charitable institution
- [ ] Organisation has been registered for at least two years
- [ ] Organisation has at least two years' experience in the social sector
- [ ] Organisation targets senior citizens as beneficiaries
- [ ] For IPSrC projects in Gap districts: Organisation has verified GAP district status with District Administration
- [ ] Organisation is applying for one of the following project types: Senior Citizen Home for 25 beneficiaries (SrCH-25), Senior Citizen Home for 50 beneficiaries (SrCH-50), or Continuous Care Home for Alzheimer's/Dementia patients (CCH) for 20 beneficiaries
- [ ] Organisation has registered on NGO-DARPAN portal and obtained NGO Unique ID
- [ ] Organisation has valid PAN card
- [ ] Organisation has valid bank account details
- [ ] Organisation has audit reports of previous years
- [ ] Organisation is applying exclusively through the e-Anudaan portal (https://grants-msje.gov.in/ngo-login)

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Organisation's PAN card
- Certificate of Registration
- Bank account details (cancelled cheque or bank statement)
- Audit reports of previous years
- Udyam Registration (if applicable)
- GST registration certificate (if applicable)
- Income Tax Returns (ITR) of previous years
- NGO Unique ID from NGO-DARPAN portal

### Scheme-Specific Documents
- Project proposal as per prescribed proforma (for SrCH-25, SrCH-50, or CCH)
- Details of infrastructure and facilities
- UDID number or UDID enrollment ID with disability certificate (if applicable under related schemes)
- For IPSrC projects in Gap districts: Verification from District Administration confirming GAP district status

### Document Quality Check
- All documents must be legible scans or clear photographs
- File names must match document type (e.g., "PAN_Card.pdf", "Registration_Certificate.pdf")
- File size must not exceed 5 MB per document
- Scanned documents must be in PDF or JPEG format
- All certificates must be valid and not expired
- Organisation name must match exactly across all documents
- Documents must be self-attested by authorised signatory
- Hindi or English language only; if in regional language, certified translation required

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
A detailed project proposal as per prescribed proforma is required for this scheme. The proposal must include:
- Executive summary
- Organisation background and experience
- Project objectives aligned with AVYAY goals
- Detailed description of proposed facility (SrCH-25, SrCH-50, or CCH)
- Beneficiary details and selection criteria
- Infrastructure plan (building, rooms, facilities)
- Staffing plan and qualifications
- Budget breakdown (recurring and non-recurring costs)
- Implementation timeline
- Monitoring and evaluation plan
- Sustainability plan post-grant period
- Annexures: Maps, photographs, cost estimates, staff CVs

### Application Form Filling Protocol
1. **Login Credentials:** Use only the username/password obtained after NGO-DARPAN and e-Anudaan registration
2. **Accuracy:** Ensure all organisational details (name, address, PAN) match exactly across documents and form fields
3. **Project Type:** Select correct project category (SrCH-25, SrCH-50, or CCH) and sub-scheme (IPSrC/AVYAY)
4. **Document Upload:** Upload documents in prescribed order and format as specified in the portal checklist
5. **Declaration:** Authorised signatory must digitally sign the declaration form
6. **Preview:** Use preview function to verify all entries before final submission
7. **Deadline Awareness:** Submit well before portal closing time (11:59 PM on deadline date) to avoid technical issues
8. **Reference Number:** Note down application reference number immediately after submission

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take screenshot of submission confirmation page showing application reference number and timestamp
- Save submission receipt PDF if generated by portal
- Store application reference number in case file with label: "AVYAY_IPSrC_AppRef_[CaseID]"
- Set calendar reminder for first follow-up at 15 days post-submission
- Set calendar reminder for status check at 30 days post-submission
- Notify client of successful submission with reference number and next steps

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and verify approval letter specifying sanctioned amount and project duration
- [ ] Confirm receipt of first installment of Grant-in-Aid in designated bank account
- [ ] Review sanction letter for any specific conditions or timelines
- [ ] Initiate project setup as per approved timeline (infrastructure, staffing, procurement)
- [ ] Maintain detailed records of all expenditures against sanctioned budget
- [ ] Prepare utilization certificates as per GFR 2017 rules
- [ ] Submit progress reports as required by the scheme guidelines
- [ ] Coordinate with nodal officer for any required inspections or verifications
- [ ] Ensure compliance with audit requirements
- [ ] Plan for subsequent installments based on utilization and reporting
- [ ] Calculate and invoice success fee as per consultancy agreement (typically 10-15% of sanctioned grant amount upon receipt of funds)
