# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Beneficiary belongs to Scheduled Caste Community  
- [ ] Applicant is an Individual, Partnership Firm, or Co-operative Society  
- [ ] For Partnership Firms/Co-operative Societies: All members belong to Scheduled Caste community  
- [ ] Annual family income of each member/applicant does not exceed Rs. 5.00 lakh (for Credit Based Schemes, effective 07th January 2026)  
- [ ] No income criterion applies for Skill Development Training Programmes  
- [ ] Application is submitted through State Channelizing Agencies (SCAs) or Channelizing Agencies (CAs); direct contact with NSFDC is not entertained  
- [ ] Eligibility verification is the responsibility of SCAs/CAs, though NSFDC may re-verify  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Incorporation/Registration Certificate  
- PAN Card  
- Income Tax Returns (ITR)  
- GST Registration (if applicable)  
- Bank Statements (last 6 months)  
- Udyam Registration (for MSMEs)  
- Aadhaar Card of authorized signatory/beneficiary  

### Scheme-Specific Documents
- Loan sanction letter (NSFDC schemes)  
- Aadhaar Card  
- Bank Passbook  
- Voter ID  
- PIA letter + Artisan Card (for Cluster beneficiaries)  
- Partner Agency letter + SHG name (for SEED beneficiaries)  
- Product photographs (minimum three)  
- Brief beneficiary and product description  
- Nomination form (prescribed format)  

### Document Quality Check
- File size: Each document < 5 MB  
- File naming: Clear, descriptive names (e.g., "Aadhaar_BeneficiaryName.pdf")  
- All documents must be legible scans or certified copies  
- Names on documents must match exactly across all submissions  
- Hindi/English documents accepted; translations required if in other languages  
- Declarations and forms must be signed and dated

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
- **DPR is not explicitly required** for standard NSFDC loan schemes based on evidence.  
- However, for complex projects (e.g., Term Loan > Rs. 25 lakh), a detailed project report may be sought by SCAs during appraisal.  
- **Verify with nodal officer (SCA/CA)** if a DPR is needed for your specific loan amount and activity.  
- If required, include: project description, cost breakdown, market analysis, technical feasibility, financial projections (5-year), repayment schedule, and risk assessment.  

### Application Form Filling Protocol
1. **Apply only through SCAs/CAs** – do not attempt direct submission to NSFDC (per eligibility criteria).  
2. **Confirm scheme type** (Term Loan, Micro Finance, Educational Loan, etc.) before filling form – limits and terms vary.  
3. **Enter accurate beneficiary details**: Name, Aadhaar, address, caste certificate (SC), and annual family income (< Rs. 5 lakh).  
4. **Specify exact project cost and loan amount requested** – ensure loan does not exceed 90% of project cost or scheme-specific cap (e.g., Rs. 45 lakh for Term Loan).  
5. **Declare no other monetary benefits** if applying for NFSC fellowship (if applicable).  
6. **Attach all scheme-specific documents** as listed in Stage 2.  
7. **Sign and date the declaration** affirming eligibility and truthfulness of information.  
8. **Retain a copy** of the submitted application and acknowledgment receipt from SCA/CA.  

### Internal Review Gate
- [ ] All applicant details complete and accurate (name, Aadhaar, income, caste)  
- [ ] Project cost and loan amount within scheme limits (e.g., Term Loan: >Rs. 1.25 lakh & ≤ Rs. 45 lakh)  
- [ ] Annual family income ≤ Rs. 5.00 lakh (for credit schemes)  
- [ ] All universal and scheme-specific documents attached  
- [ ] Documents legible, named correctly, and within size limits  
- [ ] Declarations signed and dated  
- [ ] Application routed through SCA/CA – no direct NSFDC submission attempted  
- [ ] Nomination form completed (if applicable for exhibitions/cluster/SEED)

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Obtain signed acknowledgment/receipt from SCA/CA District Office upon submission  
- Screenshot or scan the acknowledgment (if digital) or file the physical receipt  
- Note the **application reference number** provided by SCA/CA  
- Set follow-up reminders:  
  - Day 25: Check status with SCA/CA  
  - Day 35: Escalate if no update  
  - Day 45: Follow up on appraisal status

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Receive and verify **Approval Letter / Letter of Intent (LOI)** from NSFDC via SCA/CA  
- [ ] Confirm sanctioned amount matches requested loan and is within scheme limits (e.g., ≤ Rs. 45 lakh for Term Loan)  
- [ ] Verify terms & conditions: interest rate (to SCA: 2.5–5% p.a.; to beneficiary: 6.5–15% p.a.), repayment period, moratorium  
- [ ] Ensure fulfillment of **Prudential Norms** (e.g., no overdues to NSFDC, adequate government/bank guarantee)  
- [ ] Track disbursement: NSFDC → SCA/CA → Beneficiary (on demand from SCA/CA)  
- [ ] Monitor utilization: Ensure funds used only for approved income-generating activity  
- [ ] Prepare for **Utilization Certificate** submission post-expenditure (as required by SCA/CA/NSFDC)  
- [ ] Confirm repayment schedule with SCA/CA (quarterly instalments, with moratorium if applicable)  
- [ ] Invoice success fee as per consultancy agreement upon successful disbursement
