# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant belongs to DNT (De-notified Tribes), NT (Nomadic Tribes), or SNT (Semi-Nomadic Tribes) community  
- [ ] Total family income from all sources is Rs. 2.50 lakh or less per annum  
- [ ] Applicant is not availing any similar benefits from Central or State Government schemes  
- [ ] For coaching component: Family income ≤ Rs. 8 lakh per annum (as per SEED Guidelines)  
- [ ] For coaching component: Applicant has secured minimum marks required to appear for the competitive exam (e.g., 75% in 12th for JEE)  
- [ ] For livelihood component: Applicant is part of a Self-Help Group (SHG) or willing to join one  
- [ ] Applicant has valid Aadhaar card linked to bank account for DBT transfers  
- [ ] Applicant has not availed SEED scheme benefits more than twice (for coaching)  
- [ ] Applicant is willing to submit an affidavit declaring no availing of similar central/state scheme benefits  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Aadhaar Card  
- PAN Card  
- Income Tax Returns (ITR)  
- GST Certificate (if applicable)  
- Bank Statements (last 6 months)  
- Udyam Registration (for MSMEs)  
- Passport size photographs  

### Scheme-Specific Documents
- Income Certificate (issued by Revenue Officer not below Tehsildar rank)  
- Caste/Community Certificate (issued by competent authority)  
- Bank Account details (with Aadhaar linkage proof)  
- Educational qualification marksheets/certificates  
- Domicile Certificate  
- Disability certificate (if applicable)  
- Admission Rank Proof (for college scholarships)  
- Proof of fee details (for college scholarships)  
- Self-Declaration of Caste  
- Date of Birth proof  
- Endorsement by Local Pradhan of the Community (optional)  
- PAN Card of Organisation (for NGO grants)  
- PAN and Aadhar Card of Office Bearers (for NGO grants)  

### Document Quality Check
- File size: PDFs under 5 MB, images under 2 MB  
- File naming: Use format `SchemeName_ApplicantName_DocumentType.pdf` (e.g., `SEED_JyotiDevi_IncomeCertificate.pdf`)  
- All documents must be self-attested or gazetted officer-attested  
- Language: Hindi or English; vernacular documents must be accompanied by certified translation  
- Certificates must be issued within last 6 months (except caste/domicile which can be older if valid)  
- Bank account must be seeded with Aadhaar and show recent activity  

### Client Communication Template
**Subject:** Action Required: Document Submission for SEED Scheme Application – Deadline Alert  

Dear [Client Name],  

This is regarding your application for the **SEED Scheme for DNTs** (Scheme for Economic Empowerment of De-notified, Nomadic and Semi-Nomadic Tribes) implemented by the Development and Welfare Board for De-notified, Nomadic and Semi-Nomadic Communities (DWBDNC).  

To proceed, please submit the following documents via the portal **https://dwbdnc.dosje.gov.in/register** within the next 7 days:  
- Aadhaar Card  
- Income Certificate (family income ≤ Rs. 2.5 lakh/annum)  
- Caste/Community Certificate  
- Bank Account details (Aadhaar-linked)  
- Educational certificates (if applying for coaching)

- Only **online applications** are accepted; offline applications will be rejected.  
- Ensure your bank account is Aadhaar-seeded for Direct Benefit Transfer (DBT).  
- You must submit an affidavit that you have not availed benefits from any similar central or state scheme.

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not required for individual beneficiary applications under SEED. For NGO or SHG-based livelihood projects, a brief project note is advised. **Verify with nodal officer** if a Detailed Project Report (DPR) is needed for group-based interventions.

### Application Form Filling Protocol
1. Access the portal: **https://dwbdnc.dosje.gov.in/register**  
2. Select the correct component: Livelihood, Health, Housing, or Education (coaching)  
3. Fill all fields accurately; mismatched names across documents will lead to rejection  
4. Upload documents in prescribed format (PDF/JPEG, under size limits)  
5. Declare income ≤ Rs. 2.5 lakh/annum (or ≤ Rs. 8 lakh for coaching)  
6. Tick the self-declaration box for not availing similar central/state scheme benefits  
7. Provide active mobile number and email for OTP verification and communication  
8. Review all entries before final submission; no edits allowed after submit  

### Internal Review Gate
- [ ] All personal details match across Aadhaar, bank, and certificates  
- [ ] Income certificate shows family income ≤ Rs. 2.5 lakh/annum  
- [ ] Caste certificate confirms DNT/NT/SNT status  
- [ ] Bank account is Aadhaar-linked and active  
- [ ] Required scheme-specific documents uploaded (e.g., admission proof for coaching)  
- [ ] Declarations signed: no similar scheme benefits availed  
- [ ] Application submitted via online portal only (offline = auto-reject)  
- [ ] Screenshot of submission confirmation saved with reference number

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions

- [ ] Set up utilization tracking for funds received  
- [ ] Prepare Utilization Certificate (UC) as per GFR 2017 format for submission  
- [ ] Coordinate with client for success fee invoicing post-verification of fund utilization  
- [ ] Monitor for second installment release (e.g., 50% fee reimbursement after joining institute for coaching)  
- [ ] Ensure compliance with benefit caps (e.g., coaching benefits not availed more than twice)
