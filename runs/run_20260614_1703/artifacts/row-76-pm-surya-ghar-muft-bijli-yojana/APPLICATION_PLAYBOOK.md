# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is a residential household (individual homeowner or tenant with electricity connection in their name)
- [ ] Applicant has a valid electricity consumer account number from a recognized distribution utility (DISCOM)
- [ ] Applicant possesses a valid mobile number registered with the electricity consumer account
- [ ] Applicant has a valid email ID for communication and OTP verification
- [ ] Applicant can provide full name as per electricity consumer records
- [ ] Applicant can provide complete residential address including state, district, and PIN code
- [ ] Applicant agrees to have read and understood the guidelines of PM Surya Ghar: Muft Bijli Yojana (mandatory declaration during login)
- [ ] Applicant intends to install a rooftop solar system (capacity eligibility: any size, but subsidy capped at Rs. 78,000 for systems above 3 kW)
- [ ] Applicant is not applying through a vendor who will submit repeated module/inverter serial numbers based on undertaking (provision discontinued effective 26 May 2026)
- [ ] For special states (as notified by MNRE), applicant confirms eligibility for additional 10% subsidy per kW

> **Note**: There is no minimum or maximum turnover, employee count, or age of business requirement as this scheme targets individual residential households. Sector restriction: only residential (not commercial, industrial, or agricultural).

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- [ ] PAN card of the applicant (for identity and bank linkage)
- [ ] Aadhaar card of the applicant (for identity verification)
- [ ] Latest electricity bill (to verify consumer account number, name, address, and DISCOM details)
- [ ] Bank account details (cancelled cheque or bank passbook) for subsidy disbursement
- [ ] Mobile number linked to electricity consumer account (for OTP verification)
- [ ] Email ID (for communication and portal login)
- [ ] Udyam Registration: **Not applicable** (scheme is for residential households, not enterprises)

### Scheme-Specific Documents
- [ ] Valid electricity consumer account number (mandatory for fetching utility details on portal)
- [ ] Proof of residential ownership or tenancy agreement (if applicant is not the property owner, authorization from owner required for installation)
- [ ] Rooftop ownership/usage consent (if applicable in multi-ownership buildings or societies)
- [ ] No Objection Certificate (NOC) from housing society or RWA (if applicable in apartments or gated communities)
- [ ] Structural safety certificate of rooftop (may be required by empanelled vendor prior to installation, though not for initial application)
- [ ] Details of proposed solar plant capacity (kW) and estimated cost (for internal planning; not uploaded but declared in application)

> **Note**: Unlike other schemes, PM Surya Ghar does not require ITR, GST returns, audited financials, or project reports at application stage. The application is consumer-data centric.

### Document Quality Check
- [ ] All documents are clear, legible scans or photographs (minimum 300 DPI if scanned)
- [ ] File size does not exceed 2 MB per document
- [ ] File names follow format: `SchemeName_DocumentType_ApplicantName_Date.pdf` (e.g., `PMSG_MBY_ElectricityBill_RameshKumar_20260615.pdf`)
- [ ] Name on electricity bill, PAN, Aadhaar, and bank account must match exactly
- [ ] Address on electricity bill must match the address provided in the portal profile
- [ ] Mobile number must be active and capable of receiving OTPs via SMS
- [ ] Email ID must be valid and accessible
- [ ] Documents must be in English or Hindi; if in regional language, attach certified translation
- [ ] Electricity bill must be latest (not older than 3 months)

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
- **Not required** for individual residential household applications under PM Surya Ghar: Muft Bijli Yojana.  
- The application is based on consumer profile and electricity account details; no technical project report is mandated at submission stage.  
- **Verify with nodal officer**: If applying through a vendor for a GHS/RWA common facility (up to 500 kW), a preliminary feasibility report may be advised — confirm with MNRE helpline or state nodal agency.

### Application Form Filling Protocol
1. **Access the portal only via the official URL**: https://pmsuryaghar.gov.in/ — avoid third-party links or intermediaries claiming faster processing.  
2. **Use a valid, active mobile number** linked to the electricity consumer account for OTP-based login — do not use temporary or unverified numbers.  
3. **Enter profile details exactly as per electricity bill**: Name, address, state, district, and PIN code must match the utility records to avoid rejection during 'Fetch Details' step.  
4. **Select correct DISCOM/utility**: Inaccurate selection will lead to failed data fetch and application blockage — verify from electricity bill.  
5. **Enter consumer account number correctly**: Double-check for typos; this is critical for fetching consumer details and downstream subsidy processing.  
6. **Declare guideline acceptance**: Mandatory to tick “Yes, I have read all the guidelines of PM Surya Ghar Muft Bijli Yojana” before proceeding — failure to do so blocks login.  
7. **Do not submit applications with repeated module/inverter serial numbers based on vendor undertakings**: This provision is discontinued effective 26 May 2026 — ensure vendor provides unique serial numbers for all components.  
8. **Save profile before proceeding**: After entering profile details (Step 05), click ‘Save’ — failure to save will require re-entry after login timeout.

- [ ] Correct DISCOM/utility selected  
- [ ] Proposed solar capacity indicated (for internal tracking; not a portal field but advised for vendor coordination)  
- [ ] Guideline acceptance checkbox ticked during login

- [ ] Reference number noted after application initiation (if applicable)  
- [ ] Client informed of next steps: vendor selection, installation, and post-installation verification for subsidy claim

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Receive and verify approval letter from DISCOM or MNRE portal (post-installation verification)  
- [ ] Confirm sanctioned subsidy amount matches eligibility:  
    - Rs. 30,000/kW for first 2 kW  
    - Rs. 18,000/kW for next kW (up to 3 kW)  
    - Capped at Rs. 78,000 for systems above 3 kW  
    - +10% if in special state  
- [ ] Ensure installation completion certificate is issued by empanelled vendor and countersigned by DISCOM  
- [ ] Verify net metering agreement (if applicable) is signed and active  
- [ ] Confirm receipt of subsidy in bank account (disbursed after installation verification)  
- [ ] Validate that beneficiary is receiving up to 300 units of free electricity monthly (check electricity bill post-installation)  
- [ ] Guide client on utilization certificate (if required for audit — typically not for individual consumers, but retain installation invoice and commissioning report)  
- [ ] Confirm success fee invoice is raised only after subsidy receipt in client’s account  
- [ ] Advise client on annual maintenance and performance monitoring of rooftop solar system
