# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is a farmer (individual or collective farming entity)
- [ ] Applicant seeks to install standalone solar pumps (Component B)
- [ ] Applicant seeks to solarize existing grid-connected agricultural pumps (Component C)
- [ ] Applicant seeks to install grid-connected solar power plants up to 2MW on barren/fallow land (Component A)
- [ ] Application is submitted through designated departments of the State Government (not directly via central portal)
- [ ] Applicant has verified they are not using fraudulent websites or mobile applications impersonating PM-KUSUM (e.g., www.kusumyojanaonline.in.net, www.pmkisankusumyojana.co.in, www.onlinekusamyojana.org.in, www.pmkisankusumyojana.com)
- [ ] Applicant understands subsidy is up to 30% or 50% of total cost depending on component and state-specific guidelines
- [ ] Applicant has barren/fallow land available (for Component A) or existing agricultural pump to solarize (for Components B & C)
- [ ] Applicant is prepared to sell electricity to local DISCOM at state-determined tariff (for grid-connected systems under Components A & C)
- [ ] Applicant has no outstanding dues or blacklisting with state nodal agencies for renewable energy schemes

> **Note**: Eligibility is implemented by State Government departments; applicants must approach designated state-level authorities. Verify state-specific criteria with nodal officer.

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- [ ] Applicant’s PAN card
- [ ] Aadhaar card of applicant
- [ ] Bank passbook or cancelled cheque (for subsidy transfer)
- [ ] Land ownership documents (7/12 extract, mutation entry, or title deed)
- [ ] Caste certificate (if applicable for SC/ST benefits)
- [ ] Domicile certificate
- [ ] Passport-sized photographs
- [ ] Mobile number and email ID
- [ ] Udyam Registration (if applicable for agri-processing or allied activities)
- [ ] GSTIN (if registered under GST)
- [ ] Last 2 years’ Income Tax Returns (ITR) or Form 16 (if applicable)
- [ ] Bank statement of last 6 months

### Scheme-Specific Documents
- [ ] Detailed project report (DPR) or cost estimate from empaneled vendor (for solar pump or power plant)
- [ ] No Objection Certificate (NOC) from local DISCOM (for grid-connected systems - Components A & C)
- [ ] Certificate of barren/fallow land from revenue officer (Tehsildar/Mandal Revenue Officer) - **mandatory for Component A**
- [ ] Existing pump details and electricity bills (for solarization - Component C)
- [ ] Affidavit declaring land is barren/fallow and not under any dispute or litigation
- [ ] Vendor quotation from MNRE-empaneled supplier (for solar pumps/modules)
- [ ] Self-declaration form as per state-specific PM-KUSUM guidelines
- [ ] Land use conversion certificate (if converting agricultural to non-agricultural for solar plant)
- [ ] Proof of no existing subsidy availed for same purpose under other central/state schemes

### Document Quality Check
- [ ] All documents are legible, scanned in PDF format (max file size: 5 MB per document)
- [ ] Names on all documents match exactly as per Aadhaar/PAN
- [ ] Land documents are duly certified by competent revenue authority
- [ ] Bank details are IFSC-enabled and match beneficiary name
- [ ] Affidavits and declarations are notarized on stamp paper of adequate value
- [ ] Vendor quotations include GSTIN, PAN, and validity period
- [ ] DISCOM NOC is on official letterhead and signed by authorized officer
- [ ] All documents are in Hindi or English; vernacular documents must be accompanied by certified translation
- [ ] No overwriting, tampering, or use of correction fluid on physical documents

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
- **DPR Required**: Yes, for all components (A, B, C)  
- **Standard Sections**:  
  1. Executive Summary  
  2. Beneficiary Details (name, address, land details, bank info)  
  3. Technical Specification (solar pump capacity, panel wattage, structure type, inverter details)  
  4. Cost Breakdown (equipment, installation, transportation, taxes)  
  5. Implementation Timeline  
  6. Expected Outcomes (water savings, diesel reduction, income enhancement)  
  7. Annexures: Land documents, vendor quotes, DISCOM NOC, affidavits  

> **Note**: Format may vary by state. Verify exact DPR template with nodal officer or state implementing agency.

### Application Form Filling Protocol
1. **Use only official forms** provided by the State Government’s designated department (not third-party sites).  
2. **Fill in block letters**; avoid overwriting or use of whitener.  
3. **Match names exactly** as per Aadhaar and PAN across all fields.  
4. **Enter correct bank account details** (account number, IFSC, bank name) for DBT subsidy transfer.  
5. **Specify correct component** (A, B, or C) and capacity (e.g., 3HP pump, 1MW solar plant).  
6. **Declare no duplicate benefit** availed under any other central/state solar or agricultural scheme.  
7. **Attach self-attested copies** of all documents; originals to be produced only if requested.  
8. **Declare awareness of fraudulent portals** and confirm application is submitted via official state channel.  

> **Portal Note**: The central portal (https://pmkusum.mnre.gov.in) provides information and status tracking but **does not host the application form**. Applications are submitted offline/online via state portals or designated offices.

### Internal Review Gate
- [ ] All personal details (name, DOB, address) match Aadhaar/PAN  
- [ ] Bank details are correct and enabled for DBT  
- [ ] Land documents establish clear title and possession  
- [ ] Vendor quotation is from MNRE-empaneled supplier and includes warranty  
- [ ] DISCOM NOC is valid and specifies interconnection terms  
- [ ] Affidavits are notarized and on appropriate stamp paper  
- [ ] Form is signed and dated by applicant  
- [ ] All documents are attached in correct order and labeled  
- [ ] Declaration of no fraudulent website engagement is included  
- [ ] Photocopies are legible and certified where required

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- [ ] Submit application and documents to the designated office of the State Government’s nodal department (e.g., State Renewable Energy Department, Agriculture Department)  
- [ ] Obtain computer-generated receipt or acknowledgment slip with **Application Number/Reference ID**  
- [ ] Screenshot or scan the acknowledgment and store in case file with timestamp  
- [ ] Note down name, designation, and contact of the receiving officer  
- [ ] Set follow-up reminder for 15 days post-submission  
- [ ] If submitting online via state portal, save confirmation PDF and application reference number

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Collect and verify approval letter/sanction order from state nodal agency  
- [ ] Confirm sanctioned subsidy amount and total project cost  
- [ ] Ensure vendor empaneled with MNRE is selected for execution (if not already done)  
- [ ] Monitor installation progress and verify compliance with technical specifications  
- [ ] Obtain installation completion certificate from vendor and beneficiary sign-off  
- [ ] Apply for net metering/grid connection approval from DISCOM (for grid-connected systems)  
- [ ] Submit Utilization Certificate (UC) and expenditure statement to state nodal agency  
- [ ] Verify subsidy disbursement to beneficiary’s bank account via DBT  
- [ ] Guide client on selling electricity to DISCOM at state-determined tariff (for Components A & C)  
- [ ] Collect success fee invoice and close case post-utilization certificate acceptance  
- [ ] Advise client on annual maintenance and performance monitoring requirements  

> **Note**: Utilization Certificate format and timelines are state-specific. Confirm with nodal officer.
