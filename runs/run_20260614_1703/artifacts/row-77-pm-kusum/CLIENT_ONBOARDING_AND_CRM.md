## CRM Stage Pipeline

(Standard 10-stage pipeline — Lead through Disbursed/Rejected.

| Stage | Description | Action |
|---|---|---|
| 01 - Lead | Initial contact made | Schedule discovery call |
| 02 - Qualified | Eligibility confirmed for PM-KUSUM | Send proposal |
| 03 - Engaged | Retainer signed | Start onboarding |
| 04 - Docs collection | Gathering documents | Chase missing items |
| 05 - In preparation | Drafting application on https://pmkusum.mnre.gov.in/ | Internal review – Ensure DPR includes technical specifications, cost breakdown, and compliance with MNRE guidelines for Module Mounting Structure (MMS) and Domestic Content Requirement (DCR) as per OM dated 30.05.2025; coordinate with empaneled vendor for accurate quotation. |
| 06 - Submitted | Application filed on https://pmkusum.mnre.gov.in/ | Begin follow-up – Confirm receipt with State Nodal Agency; track application ID. |
| 07 - Under review | With Designated departments of the State Government | Monitor + respond to queries – Prepare for site inspection; clarify DCR, MMS, or escrow requirements if raised. |
| 08 - Approved | Sanction letter received | Invoice success fee – Verify subsidy percentage (30% or 50%) and CFA format as per latest OM (20260429-Revision of CFA formats under Comp C FLS). |
| 09 - Disbursed | Money received by client | Close case, request testimonial – Confirm installation completion and commissioning report submission. |
| 10 - Rejected | Application declined | Analyze, re-apply or alternate scheme – Common reasons: incomplete DPR, vendor not empaneled, land title dispute, or missing DISCOM NOC. |
