# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is an individual, farmer, institution, industry, or NGO  
- [ ] Applicant is the owner or authorized representative of the site where the biogas plant is to be installed  
- [ ] For community/institutional plants: applicant has legal ownership or long-term usage rights of the land  
- [ ] Proposed biogas plant conforms to MNRE-approved designs and specifications  
- [ ] Beneficiary is willing to contribute the beneficiary share as per scheme guidelines  
- [ ] Plant size is within eligible range: family-size (1 to 25 m³) or community/institutional (above 25 m³)  
- [ ] Site is suitable for biogas plant installation (access to feedstock, water, etc.)  
- [ ] No prior subsidy claimed for the same site under this scheme (if applicable)  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Identity proof of applicant (Aadhaar, Voter ID, etc.)  
- Bank account details (passbook or cancelled cheque)  
- PAN card  
- GST registration (if applicable)  
- Udyam Registration (for MSMEs)  
- Income Tax Returns (ITR) of last 2 years  
- Business registration/incorporation certificate  

### Scheme-Specific Documents
- Application form for biogas plant  
- Proof of land ownership or usage rights  
- Site plan and layout of the proposed biogas plant  
- Beneficiary share contribution proof (if applicable)  
- Completion report and utilization certificate after installation  
- Photographs of the installed biogas plant (before and after)  

### Document Quality Check
- All documents must be legible and in English or Hindi (self-attested translation if in regional language)  
- File size: PDFs under 5 MB each; images under 2 MB  
- File naming convention: `CaseID_DocumentType_Version.pdf` (e.g., `CASE001_LandProof_V1.pdf`)  
- All certificates must be valid and not expired  
- Photographs must be timestamped and clearly show plant installation stages  
- Bank details must match the name on identity proof

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not explicitly required for subsidy claim; however, a brief project report outlining plant size, type, cost estimate, beneficiary share, and expected outcomes is recommended. If required by State Nodal Agency, include:  
- Executive Summary  
- Project Background and Objectives  
- Technical Details (plant design, capacity, feedstock source)  
- Cost Breakdown (total cost, beneficiary share, CFA sought)  
- Implementation Timeline  
- Expected Benefits (energy, slurry, sanitation, income)  
- Monitoring and Maintenance Plan  
*Note: Verify with State Nodal Agency or KVIC if DPR is mandatory.*

### Application Form Filling Protocol
1. Download the official application form from https://mnre.gov.in/en/bio-gas/ or obtain from State Nodal Agency/KVIC  
2. Fill all fields in block letters; avoid overwriting or correction fluid  
3. Ensure applicant name matches exactly across all documents (ID, bank, land proof)  
4. Clearly specify plant size (in m³) and type (family/community/institutional)  
5. Attach all scheme-specific documents as annexures with proper labeling  
6. Declare beneficiary share contribution amount and source  
7. Provide accurate bank details for CFA disbursement (IFSC, account number, branch)  
8. Sign and date the form; get it notarized if required by State Nodal Agency  
9. Retain a copy of the submitted form for records  
10. Submit to State Nodal Agency or KVIC as per application process (not directly via portal unless specified)  

### Internal Review Gate
- [ ] Application form fully completed and signed  
- [ ] All universal documents attached and validated  
- [ ] All scheme-specific documents attached and validated  
- [ ] Beneficiary share proof aligns with plant size and category  
- [ ] Site plan is clear and includes dimensions, layout, and location  
- [ ] Photographs are relevant and show installation progress  
- [ ] Bank details are correct and match applicant name  
- [ ] No discrepancies in names, addresses, or dates across documents  
- [ ] Application submitted to correct authority (State Nodal Agency/KVIC)  
- [ ] Submission receipt or acknowledgment obtained

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Obtain acknowledgment/receipt from State Nodal Agency or KVIC upon submission  
- Screenshot or scan the acknowledgment (if digital) or file the physical receipt  
- Store application reference number (if provided) in case file  
- Set follow-up reminders for:  
  - Technical sanction (within 15 days of submission)  
  - Installation completion notice (as per agreed timeline)  
  - Inspection request (within 7 days of installation completion)  
  - CFA claim submission (after successful verification)

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Receive and verify approval letter from MNRE/State Nodal Agency  
- [ ] Confirm sanctioned CFA amount matches eligibility (₹9,000/m³ or ₹12,000/m³ for family-size; 30%/50% of benchmark cost for community/institutional)  
- [ ] Ensure CFA is disbursed in instalments as per milestone completion  
- [ ] Submit Utilization Certificate (UC) after full utilization of CFA  
- [ ] Maintain records of all expenditure and bills for audit purposes  
- [ ] Coordinate with client for success fee invoicing (as per firm policy)  
- [ ] Guide client on post-installation maintenance and slurry utilization for income generation  
- [ ] Monitor plant functionality for minimum 1 year post-commissioning (as per scheme guidelines)  
- [ ] Assist in renewal or extension requests if applicable
