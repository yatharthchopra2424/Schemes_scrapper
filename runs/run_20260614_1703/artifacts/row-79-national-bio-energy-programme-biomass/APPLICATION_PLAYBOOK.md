# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is an individual, organization, industry, or government/semi-government agency  
- [ ] Project is for setting up a bioenergy project (biomass, biogas, or waste-to-energy)  
- [ ] Application is submitted online via the Biourja portal for biomass and WTE programmes  
- [ ] For biogas programme, applicant submits online application on Biogas portal and physical application to Programme Implementing Agency under Biourja portal  
- [ ] Project must not divert CFA for any other purpose  
- [ ] Applicant agrees to submit a Performance Bank Guarantee of 5% of approved CFA  
- [ ] Applicant agrees to display signage: 'Project is funded under National Bio Energy Programme'  
- [ ] All documents submitted for CFA release are signed by applicant or authorized representative  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Incorporation Certificate  
- PAN Card  
- ITR (last 2 years)  
- GST Registration  
- Bank Statements (last 6 months)  
- Udyam Registration (if applicable)  

### Scheme-Specific Documents
- Detailed Project Report (DPR) outlining project scope, technology, and cost  
- Land ownership/lease documents  
- Technical details of biomass/WTE/biogas plant  
- Environmental clearances (if applicable)  
- Quotations for plant equipment  
- Declaration of no prior CFA for same project from other central schemes  
- Undertaking to submit Performance Bank Guarantee (5% of CFA)  
- Undertaking to display mandatory signage  

### Document Quality Check
- File size: <10 MB per document  
- File naming: ApplicantName_DocumentType_Date.pdf  
- All documents self-attested by applicant or authorized signatory  
- Documents in English or Hindi; regional languages require certified translation  
- Scanned copies legible and in colour where applicable  

### Client Communication Template
**Subject:** Application for National Bio Energy Programme - Biomass Scheme – Action Required  

Dear [Client Name],  

This is to inform you that your application for the **National Bio Energy Programme - Biomass** (Scheme ID: row-79) is being processed. The scheme, implemented by the Ministry of New and Renewable Energy (MNRE), provides Central Financial Assistance (CFA) for setting up bioenergy projects.  

**Key Details:**  
- **Application Portal:** https://biourja.mnre.gov.in  
- **In-principle approval timeline:** 100 days for CFA  
- **CFA Usage:** Must be exclusively earmarked for the project; diversion prohibited  
- **Post-approval requirements:**  
  - Performance Bank Guarantee of 5% of approved CFA  
  - Signage: 'Project is funded under National Bio Energy Programme' to be displayed prominently  
  - Final CFA instalment released after verification of actual vs. approved cost (lower amount considered)  

**Next Steps:**  
Please ensure all scheme-specific documents (as per checklist) are ready and submitted via the Biourja portal. For biogas projects, a physical copy must also be submitted to the Programme Implementing Agency.  

For queries, contact MNRE via the portal helpdesk or nodal officer.

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Yes, a Detailed Project Report (DPR) is required. Standard sections include:  
- Executive Summary  
- Project Objectives and Scope  
- Technology Description (biomass/WTE/biogas)  
- Site Details and Land Information  
- Process Flow Diagram  
- Equipment List and Specifications  
- Implementation Timeline  
- Cost Breakdown (CAPEX and OPEX)  
- Financial Viability Analysis  
- Environmental and Social Impact Assessment  
- Risk Mitigation Plan  
- Annexures (land documents, quotes, clearances)  

### Application Form Filling Protocol
1. Access the Biourja portal at https://biourja.mnre.gov.in  
2. Select the correct programme: Biomass, WTE, or Biogas  
3. Fill all fields accurately; do not leave mandatory fields blank  
4. Upload documents in prescribed format (PDF, <10 MB)  
5. Self-attest all uploaded documents  
6. Confirm declaration on no prior CFA for same project  
7. Review application thoroughly before final submission  
8. Save application reference number immediately after submission  
9. Do not submit hard copy unless specifically for biogas programme (physical to PIA)  

### Internal Review Gate
- [ ] All applicant details complete and correct  
- [ ] All universal and scheme-specific documents attached  
- [ ] Documents self-attested and legible  
- [ ] DPR includes all required sections  
- [ ] Declarations signed (no prior CFA, fund utilization, signage)  
- [ ] Application submitted via correct portal (Biourja for biomass/WTE; Biogas portal + physical for biogas)  
- [ ] Reference number saved and communicated to client

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take screenshot of submission confirmation page showing reference number and timestamp  
- Store reference number securely in case file  
- Set calendar reminder for follow-up at Day 25 (mid-point of 100-day SLA)  
- Set escalation trigger for Day 101 if no in-principle approval received

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Receive and verify approval letter from MNRE  
- [ ] Confirm sanctioned CFA amount matches eligibility  
- [ ] Arrange for Performance Bank Guarantee (5% of CFA) from scheduled bank  
- [ ] Display signage: 'Project is funded under National Bio Energy Programme' at project site  
- [ ] Utilize CFA exclusively for project execution as per approved DPR  
- [ ] Maintain records of all expenditures supported by CFA  
- [ ] Prepare for Utilization Certificate submission after project completion  
- [ ] Submit final documents for last CFA instalment (comparison of approved vs. actual cost)  
- [ ] Track milestone for success fee invoicing to client
