# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Entity is a food processing unit engaged in manufacturing
- [ ] Entity is an SME (Small and Medium Enterprise)
- [ ] Entity is engaged in innovative/organic food products OR undertaking branding and marketing activities
- [ ] Entity has valid business registration (Certificate of Incorporation/Registration)
- [ ] Entity has PAN registration
- [ ] Entity has GST registration
- [ ] Entity has Udyam registration (if applicable for MSME benefits)
- [ ] Entity maintains proper records of sales turnover (base year and incremental)
- [ ] Entity maintains records of plant and machinery investment
- [ ] Entity is prepared to submit periodic claims based on verified incremental sales and investment
- [ ] Entity agrees to undertake undertaking as per scheme guidelines
- [ ] Entity is willing to provide authorization letter from signatory

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Certificate of Incorporation / Registration
- PAN of the entity
- GST registration certificate
- Udyam registration (for MSME classification)
- Bank account details (cancelled cheque or bank statement)
- ITR (Income Tax Returns) for last 2-3 years
- Audited financial statements (Balance Sheet, P&L) for last 2-3 years

### Scheme-Specific Documents
- Details of plant and machinery investment (invoices, purchase orders)
- Sales turnover details (base year sales and incremental sales calculation)
- Product list and manufacturing process description
- Branding and marketing expenditure proof (if applicable for branding incentive)
- Authorization letter from signatory
- Undertaking as per scheme guidelines (signed declaration)

### Document Quality Check
- All documents must be legible scans or certified copies
- File names must match document type (e.g., "PAN_Card.pdf", "GST_Certificate.pdf")
- File size should not exceed 5 MB per document unless specified otherwise
- All documents must be in English or Hindi; if in regional language, provide certified translation
- Signatures and stamps must be visible and valid
- Bank statements must show account holder name matching PAN/entity name
- ITR must be filed and acknowledged by Income Tax Department

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not explicitly required for PLISFPI application as per evidence. However, applicants may need to prepare a project report detailing investment, sales projections, and incremental benefit claims. **Verify with nodal officer or Programme Management Agency (PMA) if a Detailed Project Report (DPR) is required for incentive approval.**

### Application Form Filling Protocol
1. Access the PLISFPI portal via https://mofpi.gov.in
2. Register as an applicant on the portal using valid email and mobile number
3. Fill in all entity details accurately (name, PAN, GST, address, contact)
4. Upload all required documents in the specified format (PDF/JPEG, under size limits)
5. Provide accurate base year sales and projected incremental sales figures
6. Detail investment in plant, machinery, and related infrastructure with supporting proofs
7. Specify incentive category applied for (Sales/Investment, Innovative/Organic, Branding & Marketing)
8. Declare that incremental sales are over base year (as only incremental sales qualify)
9. Undertake to submit periodic claims and undergo audit/verification as per scheme norms
10. Double-check all entries before final submission; incomplete forms will be rejected
11. Save application reference number immediately after submission
12. Ensure signatory is authorized via board resolution or authorization letter

### Internal Review Gate
- [ ] All sections of the application form are completed
- [ ] All universal and scheme-specific documents are uploaded and legible
- [ ] Sales turnover data (base year and incremental) is consistent with ITR and financials
- [ ] Investment details match invoices and bank statements
- [ ] Authorization letter is signed by authorized signatory
- [ ] Undertaking is signed and notarized if required
- [ ] Declarations regarding eligibility (SME, innovative/organic/branding focus) are accurate
- [ ] Application reference number is noted and stored securely
- [ ] No blank mandatory fields remain

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take screenshot of the submission confirmation page showing application reference number and timestamp
- Save the reference number in a secure tracker (e.g., Excel sheet with Case ID, Scheme, Ref No, Date)
- Set calendar reminders for follow-up at 15, 30, and 45 days post-submission
- Monitor portal for any queries or deficiency notices from MoFPI/PMA
- Respond to any queries within 7 days with requested documents or clarifications

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and verify approval letter from MoFPI/PMA specifying sanctioned incentive amount and duration
- [ ] Confirm the incentive structure: percentage of incremental sales and/or investment over 5 years
- [ ] Ensure agreement is signed with MoFPI or its appointed PMA for incentive disbursement
- [ ] Set up internal tracking for periodic claims (quarterly/annually) based on verified incremental sales and investment
- [ ] Maintain records as per scheme norms for audit and verification
- [ ] Prepare and submit first claim for incentives after achieving minimum threshold of incremental sales/investment
- [ ] Coordinate with auditor for claim validation as per scheme requirements
- [ ] Follow up on disbursement of approved incentives
- [ ] Track utilization of funds if any conditionalities apply (though PLISFPI is primarily incentive-based)
- [ ] Prepare for success fee invoicing as per consultancy agreement ([TO BE FILLED BY CONSULTANT])
