# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Beneficiary is identified as poor and vulnerable family based on Socio Economic Caste Census (SECC) database criteria  
- [ ] Household meets rural deprivation criteria: only one room, no adult member aged 16-59, female-headed with no adult male member, disabled members with no able-bodied adult, SC/ST household, or landless household deriving major income from manual casual labour  
- [ ] Household meets urban occupational criteria: ragpicker, beggar, domestic worker, street vendor, construction worker, or other specified categories  
- [ ] Family is resident in one of the 36 States/UTs implementing the scheme (including West Bengal as the latest addition)  
- [ ] No cap on family size or age applies  
- [ ] Pre-existing diseases are covered from day one  
- [ ] Beneficiary details are verified via SECC database for continued eligibility  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Proof of identity (Aadhaar card, Voter ID, PAN card, etc.)  
- Proof of address (ration card, electricity bill, etc.)  
- Ration card or SECC verification document  
- Passport-sized photographs  
- Mobile number for OTP verification  

### Scheme-Specific Documents
- No additional documents beyond the universal set are uniquely required for this scheme  

### Document Quality Check
- File size: Within portal limits (typically <5 MB per document)  
- File naming: Clear and consistent (e.g., `Aadhaar_JohnDoe.pdf`)  
- Certification: Self-attested copies acceptable; no notarization required unless specified by CSC/hospital  
- Language: Documents in Hindi or English; regional language documents must be accompanied by certified translation  
- Validity: ID and address proof must be current (not expired)  

### Client Communication Template
**Subject:** Action Required: Apply for Ayushman Bharat PM-JAY Health Cover (₹5 Lakhs/Year)  

Dear [Client Name],  

You are eligible to apply for the Ayushman Bharat Pradhan Mantri Jan Arogya Yojana (PM-JAY), a Government of India scheme providing **cashless health cover of up to ₹5 lakhs per family per year** for secondary and tertiary care hospitalization at any empanelled hospital across India.  

**Key Details:**  
- **Scheme:** Ayushman Bharat PM-JAY  
- **Portal:** https://pmjay.gov.in/  
- **Process:** Rolling basis — applications accepted throughout the year  
- **Next Steps:**  
  1. Verify eligibility via NHA website or Ayushman App using mobile number or ration card details  
  2. Visit nearest Common Service Centre (CSC) or empanelled hospital with:  
     - Proof of identity (Aadhaar/Voter ID/PAN)  
     - Proof of address (ration card/electricity bill)  
     - Ration card or SECC verification  
     - Passport-sized photos  
     - Mobile number for OTP  
  3. Submit documents for verification  
  4. Upon verification, receive your Ayushman Card (co-branded PVC)  
  5. Use the card for cashless treatment at any empanelled hospital  

**Important Notes:**  
- Coverage includes 1,949 procedures across 27 specialties (e.g., medical oncology, orthopaedics, urology)  
- Over 141 procedures exclusively for women  
- No cap on family size or age; pre-existing diseases covered from day one  
- Benefits portable across states

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not applicable. Ayushman Bharat PM-JAY does not require a Detailed Project Report (DPR) for individual beneficiary enrollment.  

### Application Form Filling Protocol
1. Use only the official NHA website (https://pmjay.gov.in/) or Ayushman App — avoid fraudulent portals  
2. Enter mobile number or ration card details to initiate eligibility check  
3. Ensure all personal details (name, age, gender) match exactly with identity proof  
4. Upload clear, legible scans of all required documents (identity, address, ration card, photos)  
5. Provide active mobile number for OTP verification — do not use landlord’s or relative’s number without consent  
6. Do not enter false income or occupation details — eligibility is SECC-based, not self-declared  
7. Review all entries before final submission; corrections after submission require CSC/hospital visit  
8. Save acknowledgment slip with application/reference number for tracking  

### Internal Review Gate
- [ ] All mandatory documents uploaded and legible  
- [ ] Information matches across identity, address, and ration card documents  
- [ ] Mobile number is active and accessible for OTP  
- [ ] Application submitted via official portal (https://pmjay.gov.in/) or Ayushman App  
- [ ] Acknowledgment/reference number recorded  
- [ ] Client informed of next steps (CSC/hospital visit for card generation)  

---

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Submit application via https://pmjay.gov.in/ or Ayushman App  
- Capture screenshot of submission confirmation page showing reference number  
- Store reference number securely in case file  
- Set follow-up reminder for 7 days post-submission to check card generation status  
- If no update, visit CSC or empanelled hospital with reference number and documents

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Collect Ayushman Card (co-branded PVC) from CSC or empanelled hospital  
- [ ] Verify card details: name, family ID, validity period  
- [ ] Confirm sanctioned amount: ₹5 lakhs per family per year (family floater basis)  
- [ ] Educate client on usage:  
    - Present Ayushman Card + ID proof at empanelled hospital for cashless treatment  
    - Coverage includes pre-hospitalization (3 days) and post-hospitalization (15 days) expenses  
    - No payment required at hospital for authorized treatments  
- [ ] Retain copy of card and acknowledgment for records  
- [ ] Inform client of renewal process (as per state-specific guidelines)  
- [ ] Submit utilization certificate if required by consulting firm (internal process)  
- [ ] Follow up on success fee invoicing per firm policy
