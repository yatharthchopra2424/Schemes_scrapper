## 8D. Internal Data Handling Policy

All client data shall be handled in accordance with the following security protocols:
- Encrypted cloud storage with AES-256 encryption
- Two-factor authentication (2FA) required for all system access
- Role-based access control with limited privileges
- Prohibition on use of personal devices for handling client data
- 2-year retention period post case closure for inactive files
- 24-hour breach notification protocol to affected clients and authorities
