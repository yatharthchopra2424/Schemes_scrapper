# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Beneficiary family comprises husband, wife, unmarried sons and/or unmarried daughters  
- [ ] Family does not own a pucca house (all-weather dwelling unit) in any part of India in the name of any family member  
- [ ] Adult earning member (irrespective of marital status) is treated as separate household if they do not own a pucca house in their name in any part of India  
- [ ] In case of married couple, either spouse or both jointly eligible for a single house, subject to household income eligibility  
- [ ] Eligible beneficiary can avail benefit under only one of the four verticals (BLC, AHP, ISSR, CLSS)  
- [ ] If beneficiary owns a pucca house with built-up area < 21 sq.m, may be eligible for enhancement up to 30 sq.m (if land/space available)  
- [ ] For CLSS vertical:  
  - [ ] EWS/LIG: Annual household income up to ₹6 lakh  
  - [ ] MIG-I: Annual household income ₹6,00,001 to ₹12,00,000  
  - [ ] MIG-II: Annual household income ₹12,00,001 to ₹18,00,000  
  - [ ] Carpet area of house ≤ 30 sq.m (EWS) or ≤ 60 sq.m (LIG) to avail interest subsidy  
  - [ ] Loan amount for interest subsidy capped at ₹6 lakh (EWS/LIG), ₹9 lakh (MIG-I), ₹12 lakh (MIG-II)  
  - [ ] Interest subsidy rates: 6.5% (EWS/LIG), 4% (MIG-I), 3% (MIG-II)  
  - [ ] CLSS vertical validity: Up to 31.03.2022 (as per evidence)  
- [ ] For BLC vertical: Central assistance of ₹1.5 lakh per house for EWS category  
- [ ] For AHP vertical: Central assistance of ₹1.5 lakh per EWS house if project has ≥35% EWS houses and ≥250 total houses  
- [ ] For ISSR vertical: Slum redevelopment grant of ₹1 lakh per house for eligible slum dwellers  
- [ ] All houses must have drinking water, power supply, sewerage, and toilet facility  
- [ ] Preference given to persons with disabilities, ST/SC/OBCs, minorities, transgender, and women (with overriding priority to widow)  

**Sources**:  
- Scheme Key Facts (Eligibility, Target Beneficiaries, Benefits)  
- https://pmay-urban.gov.in/ (About section, Eligibility details)  
- https://pmay-urban.gov.in/credit-linked-subsidy-scheme (CLSS eligibility, income limits, carpet area, loan limits)

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Identity proof (Aadhaar card, Voter ID, PAN card, etc.)  
- Income proof (salary slips, income tax returns, etc.)  
- Property details (ownership documents, encumbrance certificate, etc.)  
- Bank account details for DBT transfer  
- Affidavit declaring no pucca house ownership  
- Passport size photographs  
- Udyam Registration (if applicable for business/employment proof)  
- Latest ITR (Income Tax Return)  
- Bank statements (last 6 months)  
- GST registration certificate (if applicable)  

### Scheme-Specific Documents
- For CLSS: Loan application documents, property papers, income proof (as specified in Required Documents)  
- For BLC/AHP/ISSR: Property ownership documents, construction plan/estimate (implied from geo-tagging and stage-wise release)  
- Affidavit declaring no pucca house ownership (explicitly listed)  
- For CLSS: Documents required for loan application with Primary Lending Institutions (PLIs)  

### Document Quality Check
- All documents must be legible, scanned in PDF or clear JPEG/PNG format  
- File size per document: ≤ 5 MB  
- Name on documents must match exactly with application form and Aadhaar  
- Affidavit must be notarized on stamp paper of appropriate value  
- Income proof must be recent (last 3 months for salary slips, last FY for ITR)  
- Property documents must be clear and show ownership/encumbrance status  
- Bank account details must include IFSC, account number, and beneficiary name matching application  
- Photographs must be recent, passport-sized, with white background  
- All documents must be in English or Hindi; vernacular documents require certified translation

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
DPR is not explicitly required for individual beneficiary applications under PMAY-U. However, for AHP vertical projects (involving private/public entities), a Detailed Project Report (DPR) may be needed. Since this playbook focuses on individual beneficiary applications, **verify with nodal officer** whether DPR is required for your selected vertical (especially AHP or ARH).  

### Application Form Filling Protocol
1. Access the application portal at https://pmay-urban.gov.in/ and click on 'Apply for PMAY-U 2.0'  
2. Fill in personal and household details exactly as per identity documents (Aadhaar, PAN)  
3. Ensure beneficiary family definition matches: husband, wife, unmarried sons/daughters; adult earning members treated separately if eligible  
4. Select only **one vertical** (BLC, AHP, ISSR, or CLSS) — availing benefits under multiple verticals is prohibited  
5. For CLSS applications: Apply through Primary Lending Institutions (PLIs) with loan application and required documents; do not apply directly on PMAY-U portal for CLSS  
6. Upload documents in prescribed format (PDF/JPEG, ≤5 MB per file) with clear, legible scans  
7. Verify geo-tagging readiness: Ensure property coordinates are available for BLC/ISSR/AHP (if applicable) for stage-wise DBT release  
8. Double-check that affidavit of no pucca house ownership is signed and notarized  
9. Confirm bank account details are accurate for DBT transfer (where applicable)  
10. Save application ID and note down reference number for tracking  

**Sources**:  
- Application Process: https://pmay-urban.gov.in/ (Steps 1–5)  
- CLSS Process: Apply through PLIs, track via CLAP at https://pmayuclap.gov.in/  
- Portal Instructions: https://pmay-urban.gov.in/credit-linked-subsidy-scheme (CLSS application via PLIs)  

### Internal Review Gate
- [ ] All personal and household details filled correctly and match identity proofs  
- [ ] Selected only one vertical (BLC/AHP/ISSR/CLSS)  
- [ ] Income proof meets eligibility criteria for chosen vertical  
- [ ] Property documents show no pucca house ownership (or eligibility for enhancement if applicable)  
- [ ] Affidavit of no pucca house ownership is present, signed, and notarized  
- [ ] Bank account details are correct and match beneficiary name  
- [ ] All uploaded documents are legible, ≤5 MB, and named appropriately  
- [ ] Application ID generated and saved  
- [ ] For CLSS: Documents submitted to PLI, not directly on PMAY-U portal (unless specified)  
- [ ] Declarations and consent boxes are signed/checked

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Submit application via https://pmay-urban.gov.in/ (for BLC/AHP/ISSR) or through PLI (for CLSS)  
- Take screenshot of confirmation page showing Application ID and timestamp  
- Store Application ID/reference number in case file with client name and scheme details  
- Set calendar reminders for follow-up at 15, 30, and 45 days post-submission  
- For CLSS: Monitor status via CLSS Awas Portal (CLAP) at https://pmayuclap.gov.in/ using Application ID and OTP  
- Enable SMS alerts via registered mobile number for application status updates

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Receive and verify approval letter/sanction order from ULB/CNA  
- [ ] Confirm sanctioned amount matches eligibility (e.g., ₹1.5 lakh for BLC/AHP EWS, interest subsidy amount for CLSS)  
- [ ] For BLC/ISSR/AHP: Ensure geo-tagging is completed at each construction stage for DBT release  
- [ ] For CLSS: Confirm subsidy amount credited upfront to loan account via PLI; verify revised EMI and loan statement  
- [ ] Utilization Certificate (UC) submission: Required for stage-wise fund release (BLC/ISSR/AHP) or as per CNA guidelines  
- [ ] Invoice success fee as per consultancy agreement  
- [ ] Advise beneficiary to retain all approval documents, sanction letters, and UC for future reference  
- [ ] For CLSS: Monitor loan account for subsidy credit and ensure no duplicate claims  

**Sources**:  
- Financial Support: Direct Benefit Transfer (DBT) mode for BLC; upfront credit to loan account for CLSS  
- Monitoring: PMAY-U MIS, CLAP, geo-tagging with PFMS and GIS  
- Approval Process: https://pmay-urban.gov.in/ (Step 6: Upon approval, applicant registered as beneficiary)  
- CLSS: https://pmayuclap.gov.in/ (subsidy credited to loan account via PLIs)  
- Utilization: Standard requirement for CSS schemes; implied from fund release tranches in sanction orders
