# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is an individual or institution holding manuscripts (citizen contributor, scholar, library, monastery, temple, academic institution, or similar)
- [ ] Applicant is located anywhere in India (Pan-India geographic scope)
- [ ] Applicant wishes to participate in the preservation, digitization, and sharing of manuscript heritage
- [ ] Applicant agrees to join the Gyan Bharatam Parivar network as a recognized scholar or contributing member
- [ ] Applicant has access to manuscripts for survey, conservation, digitization, or research purposes
- [ ] Applicant is willing to collaborate with researchers, scholars, and institutions across India
- [ ] Applicant accepts that manuscript conservation and digitization efforts are subject to institutional partnerships and MoUs
- [ ] Applicant understands that access to certain manuscript collections may be restricted based on custodian policies

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- PAN card of applicant (individual or authorized signatory for institution)
- Aadhaar card of applicant (individual or authorized signatory)
- Bank account details (cancelled cheque or bank statement)
- Proof of address (utility bill, rental agreement, or property tax receipt)
- Udyam Registration Certificate (if applicable for MSME institutions)
- GST Registration Certificate (if applicable)
- Latest Income Tax Return (ITR) acknowledgment (if applicable)
- Audited financial statements (for institutions, if available)

### Scheme-Specific Documents
- Manuscript inventory list (details of manuscripts held: title, script, language, material, approximate age, condition)
- Photographs of manuscripts (clear images showing text, physical condition, and any unique features)
- Description of manuscript collection (origin, provenance, historical significance, current custodianship)
- Statement of intent (outlining applicant’s goals for participation in Gyan Bharatam Mission)
- Institutional registration certificate (for libraries, monasteries, temples, academic institutions, or NGOs)
- Authorization letter (if applying on behalf of an institution, signed by competent authority)
- MoU draft or expression of interest (for potential collaboration on conservation, digitization, or research)

### Document Quality Check
- All documents must be legible, scanned in PDF or JPEG format
- File size for each document must not exceed 5 MB
- File names must clearly indicate document type and applicant name (e.g., "PAN_JohnDoe.pdf", "ManuscriptInventory_ABCLibrary.pdf")
- Photographs must be in focus, well-lit, and show manuscript details clearly
- Institutional documents must be current and valid (not expired)
- Declarations and statements must be signed and dated by the applicant or authorized signatory
- All documents must be in English or Hindi; if in another language, a certified translation must be attached

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not required. The Gyan Bharatam Mission does not mandate a Detailed Project Report (DPR) for scholar registration or manuscript contribution. Applicants are required to submit a manuscript inventory and statement of intent instead. **Verify with nodal officer if applying for institutional partnership or MoU-based projects.**

### Application Form Filling Protocol
1. Access the registration portal at **https://gyanbharatam.com/** and navigate to the “Register as Scholar” section via the homepage or community page.  
2. Fill in all personal details accurately: full name, date of birth, gender, contact information (email, phone), and residential address.  
3. Provide complete institutional details (if applicable): name, type (library, monastery, temple, academic institution, etc.), registration number, and official address.  
4. Enter manuscript collection details: total number of manuscripts, scripts, languages, materials (palm-leaf, birch bark, paper, etc.), and brief description of significance.  
5. Upload all required documents in the specified format (PDF/JPEG, <5 MB each) with clear file names.  
6. Read and accept the declaration regarding data use, manuscript sharing, and adherence to Gyan Bharatam’s objectives.  
7. Submit the form and note down the application reference number displayed on-screen.  
8. Do not refresh or navigate away from the page until submission confirmation is received.

### Internal Review Gate
- [ ] All personal and institutional fields in the application form are complete and accurate  
- [ ] Manuscript inventory list is detailed and matches uploaded photographs  
- [ ] Statement of intent is clear, signed, and aligned with Gyan Bharatam’s objectives  
- [ ] All universal and scheme-specific documents are attached and legible  
- [ ] File sizes are within limits and names follow naming convention  
- [ ] Declarations are signed and dated  
- [ ] Screenshot of successful submission has been taken and saved  
- [ ] Application reference number is recorded in the case file

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Submit the application via the official portal: **https://gyanbharatam.com/**  
- Immediately after submission, take a screenshot of the confirmation page showing the application reference number and timestamp  
- Save the screenshot in the case folder with the naming convention: “Submission_Confirmation_[CaseID]_[Date].png”  
- Record the application reference number in the case tracker and notify the client  
- Set an automated follow-up reminder for 15 days post-submission to check for approval or request for additional information

### Approval Actions

- [ ] Assist in preparing utilization reports or progress updates if required for ongoing engagement  
- [ ] Track and invoice the success fee as per the consultancy agreement with the client  
- [ ] Maintain records for potential renewal or advancement to institutional partnership/MoU under the mission
