# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is a scheduled airline operator
- [ ] Applicant qualifies through transparent, competitive bidding process for specific routes
- [ ] Applicant meets technical criteria set by MoCA and AAI (aircraft suitability, operational capability)
- [ ] Applicant meets financial criteria set by MoCA and AAI
- [ ] Applicant complies with regulatory requirements
- [ ] Applicant is eligible for zero-VGF bid or opts for VGF support if needed
- [ ] Applicant is a domestic or international airline (including code-sharing arrangements)
- [ ] Route proposal submitted through UDAN portal for specific route allocation

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Certificate of Incorporation / Registration
- PAN Card
- Income Tax Returns (ITR) for last 3 years
- GST Registration Certificate
- Bank Statements (last 6 months)
- Udyam Registration (if applicable)

### Scheme-Specific Documents
- Air Operator’s Permit (AOP)
- Aircraft Registration and Airworthiness Certificates
- Financial Statements and Bank Guarantees
- Route Proposal Document
- Bid Submission Form
- Security Clearance (if applicable)
- Undertaking for Compliance with Scheme Guidelines
- Details of Aircraft Fleet and Seating Capacity
- Operational Plan and Schedule

### Document Quality Check
- All documents must be legible and in PDF format
- File size must not exceed 10 MB per document
- Document names must match the checklist exactly (e.g., "Aircraft_Registration_Certificate.pdf")
- All certificates must be valid and not expired
- Undertakings and declarations must be signed and stamped
- Language: English or Hindi; if in regional language, certified translation required

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not explicitly required as per evidence; however, a **Route Proposal Document** and **Operational Plan and Schedule** are mandatory. Verify with nodal officer if a detailed DPR is expected for complex routes.

### Application Form Filling Protocol
1. Access the UDAN portal at https://www.aai.aero/en/rcs-udan using your AAI email credentials.
2. Select “Submit Proposal” for the target route under the current bidding cycle.
3. Fill in all fields in the Bid Submission Form with accurate data from collected documents.
4. Quote the **VGF per seat** competitively – remember, the contract is awarded to the bidder quoting the **lowest VGF per seat**.
5. Ensure the Route Proposal Document includes departure/arrival airports, proposed schedule, and aircraft type.
6. Attach all scheme-specific documents in the prescribed format and naming convention.
7. Sign the Undertaking for Compliance with Scheme Guidelines digitally or via scanned copy.
8. Double-check that the bid includes details of aircraft fleet, seating capacity, and operational capability.

### Internal Review Gate
- [ ] All application form fields completed accurately
- [ ] All universal and scheme-specific documents attached and validated
- [ ] Bid Submission Form includes VGF per seat quote
- [ ] Route Proposal Document and Operational Plan are consistent
- [ ] Undertaking for Compliance is signed and dated
- [ ] Aircraft details match registration and airworthiness certificates
- [ ] Financial statements support the bid’s viability
- [ ] Security clearance attached if applicable
- [ ] Final review by senior consultant before submission

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take a screenshot of the submission confirmation page showing the timestamp and application/reference number.
- Store the reference number securely in the case file.
- Set a follow-up reminder for 15 days post-submission to check application status.
- Monitor the portal for bidding updates and route allocation announcements.

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and approve the sanction letter from MoCA/AAI
- [ ] Verify the sanctioned VGF amount and duration (up to 10 years, indexed to ATF prices and inflation)
- [ ] Confirm eligibility for concessions: 2% excise duty on ATF for 3 years, waiver of landing/parking/TNLC, nominal RNFC
- [ ] Ensure state government commitments: free land, multi-modal hinterland connectivity, VAT on ATF ≤1%
- [ ] Prepare for utilization certificate submission as per periodic review requirements
- [ ] Track invoice for success fee as per consultancy agreement
- [ ] Monitor compliance: maintain minimum Passenger Load Factor (PLF) to continue receiving VGF
- [ ] Note exclusivity period: 3 years on awarded routes prevents competition
- [ ] Remember: only 50% of seats eligible for fare caps; remaining seats market-priced
- [ ] Cargo operators are not eligible for VGF – ensure client is a passenger operator if claiming VGF
