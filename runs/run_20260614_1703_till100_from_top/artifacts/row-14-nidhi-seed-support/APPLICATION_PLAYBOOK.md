# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] **For Incubators/TBIs**: Application for financial support will be accepted once per year through an online portal. A call for applications will be announced at the appropriate time.  
- [ ] **For Startups/Entrepreneurs**: Innovator or startup must apply to one of the listed Startup Incubators/Technology Business Incubators (TBIs) for the NIDHI Seed Support Program (NIDHI-SSP).  
- [ ] **For Startups/Entrepreneurs**: Startup Incubators/TBIs announce periodic calls for applications; innovators and startups are encouraged to visit the website of Startup Incubators regularly to stay informed about these calls.  
- [ ] **Target Beneficiaries**: Startups; entrepreneurs; students; researchers; innovators.  
- [ ] **Geographic Scope**: Pan-India.  
- [ ] **Scheme Type**: Grant.  
- [ ] **Implementing Agency**: Department of Science and Technology (DST), Government of India.  
- [ ] **Financial Support**: For Incubators/TBIs: Up to INR 1000 lakh; for Startups/Entrepreneurs: Up to INR 100 lakh.  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Incorporation Certificate  
- PAN Card  
- ITR (Income Tax Returns)  
- GST Registration  
- Bank Statements (last 6 months)  
- Udyam Registration (if applicable)  

### Scheme-Specific Documents
- **For Incubators/TBIs**: Proposal for financial support (as per guidelines), including details of the incubator’s capacity, pipeline, and utilization plan.  
- **For Startups/Entrepreneurs**: Application to the listed TBI/Incubator (no direct application to DST; must go through TBI).  
- **Supporting Documents**: Guidelines document (NIDHI Seed Support Program (NIDHI-SSP) Guidelines), list of supported TBIs, and any call-for-application notice from the TBI.  

### Document Quality Check
- File size: Under 10 MB per document.  
- File naming: Clear and consistent (e.g., `PAN_Card.pdf`, `Bank_Statement_Jan-Jun_2024.pdf`).  
- All documents must be legible, scanned in color if original is colored, and certified true copies where required.  
- Language: English or Hindi; if in another language, provide a certified translation.  
- Declarations and undertakings must be signed and dated.

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
- **DPR Required?** Not explicitly mentioned in evidence.  
- **Action**: Verify with the nodal officer or TBI whether a Detailed Project Report (DPR) is required for the specific application (incubator or startup).  

### Application Form Filling Protocol
1. **Access the Portal**: Visit the official NIDHI-SSP portal at [https://nidhi.dst.gov.in/nidhissp/](https://nidhi.dst.gov.in/nidhissp/) for guidelines and updates.  
2. **For Incubators/TBIs**: Download the proposal format from the portal (if available) or use the guidelines to structure your application.  
3. **For Startups/Entrepreneurs**: Apply directly to the listed TBI; do not apply to DST portal.  
4. **Complete All Sections**: Fill in every field; if not applicable, write “NA” or “Not Applicable.”  
5. **Attach Documents**: Ensure all universal and scheme-specific documents are attached as per the checklist.  
6. **Declaration**: Sign and date all undertakings and declarations.  
7. **Review Before Submission**: Double-check for errors, missing fields, or incorrect data.  
8. **Submit Online**: Submit via the TBI’s portal (for startups) or DST’s online portal (for incubators/TBIs, when call is open).  
9. **Save Confirmation**: Take a screenshot of the submission confirmation and save the application/reference number.  

### Internal Review Gate
- [ ] All applicant details filled correctly (name, address, contact, etc.).  
- [ ] All universal documents attached and verified.  
- [ ] All scheme-specific documents attached (proposal, guidelines, TBI list if applicable).  
- [ ] Declarations signed and dated.  
- [ ] File sizes within limits and names standardized.  
- [ ] Application/reference number noted post-submission.  
- [ ] Submission screenshot saved.  
- [ ] Follow-up reminders set for Day 18 onwards.

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take a screenshot of the submission confirmation page showing application/reference number and timestamp.  
- Store the reference number securely in the case file.  
- Set calendar reminders for follow-up at 7, 15, and 30 days post-submission.  
- Save all submitted documents in a dedicated folder labeled with Case ID and Scheme Name.

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and verify the approval letter from DST or the TBI (as applicable).  
- [ ] Confirm the sanctioned amount matches the approved request (up to INR 100 lakh for startups, up to INR 1000 lakh for incubators/TBIs).  
- [ ] Utilization Certificate: Prepare and submit as per GFR rules, detailing fund usage for seed support activities (prototype development, market entry, etc.).  
- [ ] Audit: Ensure audited statement of accounts is prepared and submitted with the Utilization Certificate.  
- [ ] Invoice Success Fee: Raise invoice for consultancy success fee as per the engagement agreement with the client.  
- [ ] Monitor Utilization: Track fund usage to ensure compliance with scheme objectives (bridging development and commercialization).  
- [ ] Report Progress: Submit periodic progress reports as required by DST or the TBI.  
- [ ] Plan for Next Steps: Advise client on post-seed support strategies (e.g., seeking follow-on funding, scaling operations).
