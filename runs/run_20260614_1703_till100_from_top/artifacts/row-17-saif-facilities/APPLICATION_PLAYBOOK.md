# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is an academic institution engaged in scientific research and development activities
- [ ] Applicant is a national laboratory engaged in scientific research and development activities
- [ ] Applicant is an industry entity engaged in scientific research and development activities
- [ ] Applicant is a research scholar affiliated with an eligible institution
- [ ] Applicant has identified a required facility or instrument from the SAIF network
- [ ] Applicant has prepared a project proposal justifying facility use
- [ ] Applicant has obtained institutional authorization for facility access
- [ ] Applicant has detailed the intended use of the facility
- [ ] Applicant is prepared to settle any applicable user charges (if applicable)

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Institutional registration certificate
- PAN card of the institution
- Income Tax Returns (ITR) for last 2 years
- GST registration certificate (if applicable)
- Bank statements (last 6 months)
- Udyam Registration (for MSMEs, if applicable)
- Audited financial statements

### Scheme-Specific Documents
- Project proposal detailing research objectives and justification for facility use
- Institutional authorization letter from competent authority
- Details of intended use of facility (specific instruments, duration, methodology)
- User charges payment proof (if applicable, after facility utilization)

### Document Quality Check
- All documents must be legible and in PDF format
- File names must match the document type (e.g., "Project_Proposal.pdf")
- Institutional authorization must be on official letterhead with seal/signature
- Project proposal must include clear objectives, methodology, and expected outcomes
- Documents must be in English or Hindi; if in another language, certified translation required
- File size for each document should not exceed 10 MB

### Client Communication Template
**Subject:** Application for SAIF Facilities Access – [Institution Name]  

Dear [Client Name],  

This communication outlines the document requirements for accessing facilities under the **SAIF Facilities** scheme implemented by the **Department of Science & Technology (DST)**.  

The SAIF Facilities scheme provides access to sophisticated analytical instruments and high-end equipment across its pan-India network. It does not provide direct financial support; instead, it facilitates shared use of existing infrastructure. Users may be charged nominal user fees for facility utilization.  

To proceed, please collect the following scheme-specific documents:  
1. A detailed project proposal justifying the need for SAIF facility access  
2. An institutional authorization letter from your organization’s competent authority  
3. Specific details of intended use (instruments, time slots, methodology)  
4. Proof of payment for user charges (if applicable, post-utilization)  

These documents must be submitted along with universal documents (PAN, ITR, GST, bank statements, Udyam, etc.) via the respective SAIF center’s application portal.  

Access is granted based on project proposals and institutional requirements, subject to equipment availability and scheduling. Prior approval from the facility in-charge is mandatory before using any SAIF facility.  

For portal access and center-specific details, visit the DST website: https://dst.gov.in

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
A detailed project report is required for this scheme. It should include:  
- Title and objectives of the research project  
- Justification for requiring SAIF facility access  
- List of specific instruments/facilities to be used from the SAIF network  
- Detailed methodology and experimental plan  
- Expected outcomes and deliverables  
- Timeline for facility utilization  
- Budget for user charges (if applicable)  
- Institutional background and research team credentials  

### Application Form Filling Protocol
1. Access the application portal of the specific SAIF center housing the required facility  
2. Select “New Application for Facility Access” or equivalent option  
3. Enter accurate institutional details (name, address, PAN, Udyam if applicable)  
4. Upload the project proposal as a single PDF file (<10 MB)  
5. Upload institutional authorization letter on official letterhead  
6. Specify the exact facility/instrument required and intended duration of use  
7. Describe the intended use of facility with technical justification  
8. Declare willingness to pay applicable user charges (if any)  
9. Review all entries for accuracy before final submission  
10. Submit and save the application reference number  

### Internal Review Gate
- [ ] All sections of the application form are completed  
- [ ] Project proposal is uploaded and clearly justifies facility need  
- [ ] Institutional authorization letter is valid and signed  
- [ ] Details of intended use are specific and technically sound  
- [ ] User charges acknowledgment is included (if applicable)  
- [ ] All universal documents are attached and legible  
- [ ] Declarations regarding prior approval requirement are acknowledged  
- [ ] Application reference number is noted post-submission

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take a screenshot of the submission confirmation page showing the application reference number  
- Store the reference number in the case file and client communication log  
- Set a follow-up reminder for 7 days post-submission to check status  
- Notify the client of successful submission with the reference number

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Receive and verify approval letter from SAIF facility in-charge  
- [ ] Confirm sanctioned facility access duration and time slots  
- [ ] Note any applicable user charges and payment schedule  
- [ ] Schedule facility access as per approved timetable  
- [ ] Utilize the facility strictly as per allocated time and guidelines  
- [ ] Submit utilization report post-access as required by the scheme  
- [ ] Settle any user charges invoiced by the SAIF center  
- [ ] Retain all approvals, utilization reports, and payment receipts for audit purposes  
- [ ] Notify client of successful facility utilization and project progress
