About Atal Incubation Centre

To create an ever-evolving ecosystem of start-ups and entrepreneurs, AIM has been establishing world class incubators called Atal Incubation Centres (AICs) at universities, institutions and corporates among others. These Centres aim to foster and support world class innovation, dynamic entrepreneurs who want to build scalable and sustainable enterprises.
