# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is an individual, self-help group, institution, or MSME entrepreneur  
- [ ] Applicant is above 18 years of age  
- [ ] For manufacturing sector projects costing > Rs. 10 lakh: applicant possesses at least VIII standard pass educational qualification  
- [ ] For business/service sector projects costing > Rs. 5 lakh: applicant possesses at least VIII standard pass educational qualification  
- [ ] Project cost does not exceed Rs. 50 lakh (manufacturing sector) or Rs. 20 lakh (business/service sector)  
- [ ] Applicant has not previously availed subsidy under any other central or state government scheme  
- [ ] Applicant is not an existing unit under PMRY, REGP, or any other government scheme  
- [ ] Project is technically feasible and economically viable  
- [ ] Applicant possesses valid identity proof (Aadhaar, Voter ID, PAN, etc.)  
- [ ] Applicant possesses valid address proof  
- [ ] Applicant has a valid bank account  

> **Sources**: Scheme Key Facts (Eligibility, Target Beneficiaries, Key Caveats), Application Portal: https://www.kviconline.gov.in/pmegpeportal/pmegphome/index.jsp  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Identity Proof (Aadhaar, Voter ID, PAN, etc.)  
- Address Proof  
- PAN Card  
- Bank Statement (last 6 months)  
- Udyam Registration Certificate (if applicable)  
- GST Registration Certificate (if applicable)  
- Income Tax Returns (last 2 years, if applicable)  
- Passport Size Photographs (2 copies)  

### Scheme-Specific Documents
- Project Report (detailed proposal for the micro-enterprise)  
- Proof of Age (Birth Certificate, School Leaving Certificate, etc.)  
- Educational Qualification Certificate (if project cost > Rs. 10 lakh in manufacturing or > Rs. 5 lakh in business/service sector)  
- Caste Certificate (for SC/ST/OBC beneficiaries seeking higher subsidy)  
- Special Category Certificate (if applicable: ex-servicemen, physically handicapped, etc.)  
- Rural Area Certificate (if seeking higher subsidy for NER, hill, or border areas)  
- Bank Account Details (cancelled cheque or passbook copy)  

### Document Quality Check
- [ ] All documents are clear, legible scans or certified copies  
- [ ] File size does not exceed 2 MB per document  
- [ ] Document names match the applicant’s identity proof exactly (no spelling variations)  
- [ ] Certificates are issued by competent authorities and are valid (not expired)  
- [ ] Documents are in English or Hindi; if in regional language, accompanied by notarized translation  
- [ ] Project Report is signed and dated by the applicant  
- [ ] Bank account details include IFSC code and account number

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
✅ **Required**. The Project Report is mandatory for all PMEGP applications.  
**Standard Sections to Include**:  
- Executive Summary  
- Promoter Details (name, age, qualification, experience)  
- Project Description (product/service, manufacturing process or service flow)  
- Market Analysis (demand, competition, target customers)  
- Marketing and Sales Strategy  
- Organizational Plan (manpower, roles)  
- Technical Details (machinery, technology, raw materials)  
- Financial Projections (cost of project, means of finance, profitability, break-even analysis, cash flow for 3 years)  
- Viability Assessment (technical, economic, financial)  
- Employment Generation Details  
- Implementation Timeline  

> **Note**: The Project Report must establish technical feasibility and economic viability to qualify for bank loan and subsidy.  

### Application Form Filling Protocol
1. Access the official PMEGP e-portal: https://www.kviconline.gov.in/pmegpeportal/pmegphome/index.jsp  
2. Select “New User Registration” if first-time user; otherwise, log in with credentials  
3. Fill applicant details exactly as per identity proof (name, DOB, address, etc.)  
4. Select beneficiary category (Individual/SHG/Institution) and special category (if applicable)  
5. Enter project details: sector (manufacturing/service), project cost, proposed activity  
6. Upload all required documents in prescribed format (PDF/JPEG, <2 MB)  
7. Declare that applicant has not availed subsidy under any other central/state scheme and is not an existing unit under PMRY/REGP  
8. Verify all entries before final submission; no edits allowed after submission  
9. Save application reference number immediately upon successful submission  
10. Forward the application to the concerned District Industries Centre (DIC) or KVIC/KVIB office for scrutiny  

### Internal Review Gate
- [ ] All applicant details match identity proof exactly  
- [ ] Project cost is within sector-specific limits (≤ Rs. 50 lakh manufacturing / ≤ Rs. 20 lakh service)  
- [ ] Educational qualification uploaded if project cost exceeds threshold  
- [ ] Caste/special category certificates uploaded for eligible beneficiaries  
- [ ] Project Report is complete, signed, and demonstrates viability  
- [ ] Bank account details are correct and active  
- [ ] Declarations regarding prior subsidy and existing unit status are correctly selected  
- [ ] All documents are uploaded, legible, and within file size limits  
- [ ] Application reference number is saved and shared with client

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Submit the application via the PMEGP e-portal: https://www.kviconline.gov.in/pmegpeportal/pmegphome/index.jsp  
- Take a screenshot of the confirmation page showing the **Application Reference Number** and timestamp  
- Store the screenshot and reference number in the case folder with naming convention: `PMEGP_Submission_[CaseID]_[Date].png`  
- Set automated reminders for follow-up at 15, 30, and 45 days post-submission  
- Inform the client of submission and provide the reference number for tracking

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and verify the approval letter from the bank or KVIC detailing sanctioned loan amount and margin money subsidy  
- [ ] Confirm the margin money subsidy (15%–35% of project cost) is credited to a special term deposit account (not withdrawable until lock-in period ends)  
- [ ] Verify loan disbursement from the bank for the balance project cost (project cost minus margin money)  
- [ ] Monitor utilization of funds and ensure compliance with scheme guidelines (no premature withdrawal of margin money)  
- [ ] Guide client on submitting Utilization Certificate (UC) and progress reports to the bank as required  
- [ ] Confirm invoice for success fee only after successful disbursement of both loan and margin money subsidy  
- [ ] Advise client on availing training and technical support through KVIC and its affiliated institutions  
- [ ] Ensure client understands that existing units or prior subsidy beneficiaries are ineligible, and this approval is contingent on continued compliance  

> **Sources**: Scheme Key Facts (Financial Support, Benefits, Application Process, Key Caveats), Application Portal: https://www.kviconline.gov.in/pmegpeportal/pmegphome/index.jsp
