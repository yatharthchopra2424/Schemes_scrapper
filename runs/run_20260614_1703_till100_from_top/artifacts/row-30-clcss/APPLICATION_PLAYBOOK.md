# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Enterprise is classified as a Micro or Small Enterprise (MSE) under MSME definition  
- [ ] Enterprise has availed institutional finance (term loan) of up to Rs 1 crore for technology upgradation  
- [ ] Technology adopted is among the 51 approved sub-sectors/products under CLCSS guidelines  
- [ ] Technology is well-established and improved, duly approved under scheme guidelines  
- [ ] Enterprise may be new (set up with eligible technology) or existing (upgrading technology)  
- [ ] Loan is a term loan, not working capital finance  
- [ ] Claim is submitted within the specified period after loan disbursement and installation (verify exact timeline with nodal agency)  
- [ ] Enterprise holds a valid UDYAM Registration Certificate  
- [ ] Enterprise has not previously claimed subsidy under CLCSS for the same unit/technology upgradation  

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- UDYAM Registration Certificate  
- PAN Card of the enterprise  
- GST Registration Certificate  
- Bank statements (last 6 months)  
- Income Tax Returns (ITR) for last 2 financial years  
- Incorporation/Registration documents (Partnership deed, MoA, CoI, etc.)  
- Cancelled cheque or bank account details  

### Scheme-Specific Documents
- Loan sanction letter from bank/financial institution  
- Proof of loan disbursement (bank statement showing credit)  
- Invoice and payment proof for machinery/technology purchased  
- Certificate of installation and commissioning  
- Details of technology adopted (specification, approval under 51 sub-sectors)  
- Authorization letter for subsidy claim (addressed to lending institution)  

### Document Quality Check
- All documents must be legible scans or clear photographs  
- File size per document: < 5 MB  
- File names must follow format: `SchemeName_DocumentType_ClientName_Date.pdf` (e.g., `CLCSS_LoanSanction_ABCPvtLtd_20240515.pdf`)  
- UDYAM, PAN, GST details must match exactly across all documents  
- Bank account details must match in loan sanction, disbursement proof, and authorization letter  
- Technology specification must clearly reference approval under one of the 51 sub-sectors  
- Installation certificate must be issued by supplier or authorized engineer  
- All documents must be in English or Hindi; if in regional language, attach certified translation

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not required for subsidy claim under CLCSS. However, a brief note on technology upgradation (including specification, cost, and approval under 51 sub-sectors) must be included in the application package. Verify with nodal officer if a detailed DPR is sought in specific cases.

### Application Form Filling Protocol
1. Ensure the applicant name matches exactly as per UDYAM and bank records  
2. Loan amount entered must not exceed Rs 1 crore (eligibility cap)  
3. Subsidy claimed = 15% of loan amount (capped at Rs 15 lakh)  
4. Technology details must explicitly state conformity with one of the 51 approved sub-sectors  
5. Date of loan disbursement and installation must be clearly mentioned  
6. Bank details (account number, IFSC) must match across loan documents and authorization letter  
7. Authorization letter must be signed by authorized signatory and addressed to the lending institution for subsidy release  
8. Do not leave any mandatory field blank; if not applicable, write “NA” with justification  

### Internal Review Gate
- [ ] All universal and scheme-specific documents attached  
- [ ] Application form filled completely with accurate figures  
- [ ] UDYAM, PAN, GST numbers match across documents  
- [ ] Loan amount ≤ Rs 1 crore and subsidy calculated correctly (15%)  
- [ ] Technology specification includes reference to approved sub-sector  
- [ ] Installation certificate is recent and signed  
- [ ] Authorization letter is on company letterhead and signed  
- [ ] Declarations signed by authorized signatory  
- [ ] File naming convention followed  
- [ ] Reference number from lending institution noted (if available)

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Submit the complete application package through the lending institution (bank/financial institution)  
- Obtain a submission receipt or acknowledgment from the bank  
- Screenshot the submission confirmation (if portal-based) and save with case ID  
- Store the application reference number provided by the lending institution  
- Set follow-up reminders:  
  - Day 25: Check status with lending institution  
  - Day 35: Escalate if no acknowledgment from nodal agency  
  - Day 45: Follow up on verification status

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain approval letter from nodal agency via lending institution  
- [ ] Verify sanctioned subsidy amount equals 15% of loan amount (capped at Rs 15 lakh)  
- [ ] Confirm subsidy has been adjusted against the loan account by the lending institution (request loan statement showing reduction)  
- [ ] Ensure utilization certificate is not required (subsidy is upfront and adjusted to loan)  
- [ ] Coordinate with client and lending institution for successful closure  
- [ ] Prepare and submit invoice for success fee as per engagement terms  
- [ ] Archive all documents with case ID for audit and future reference
