# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] MSME registered with UDYAM registration portal (MoMSME)
- [ ] Valid Udyam Registration Number
- [ ] Mobile number associated with Udyam Registration
- [ ] Valid Email ID
- [ ] GST Number (if applicable)
- [ ] TAN Number (if applicable)
- [ ] MSME Representative Selfie (to be submitted via mobile app)
- [ ] Site Exterior Photographs (to be submitted via mobile app)
- [ ] Basic Information and Documents (as per certification level requirements)
- [ ] Any number of Units registered under one Udyam Registration can apply (each Unit applies separately)
- [ ] No restrictions on registration type, turnover threshold, employee count, sector, or age of business beyond UDYAM registration

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Udyam Registration Certificate
- PAN Card of the business
- ITR (last 2 years)
- GST Registration Certificate (if applicable)
- Bank Statements (last 6 months)
- Udyam Registration Number (already verified in Stage 1)

### Scheme-Specific Documents
- MSME Representative Selfie (captured via ZED MSME mobile app)
- Site Exterior Photographs (captured via ZED MSME mobile app)
- Basic Information and Documents (as per certification level requirements – e.g., product details, quality certificates, operational processes, customer information)
- GST Number and TAN Number (if applicable, to be provided during application)
- Undertaking of Compliance (Annexure 2, to be signed online)
- ZED Pledge Certificate (generated after taking the pledge online)

### Document Quality Check
- File size: Must not exceed 2MB for uploads (as per financial assistance manual, applicable by analogy)
- File name: Must match the document type (e.g., "Udyam_Registration.pdf", "Selfie.jpg")
- Certification: Self-attested copies acceptable; no notarization required unless specified
- Language: Documents in English or Hindi; translations not required if original is in these languages
- Clarity: Photographs must be clear and legible; selfie must show the representative clearly
- Validity: Udyam Registration must be active and valid at time of application

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not required for ZED Certification application. The scheme does not mandate a Detailed Project Report (DPR). However, applicants must submit "Basic Information and Documents" as per certification level requirements via the ZED MSME mobile app. Verify with nodal officer if sector-specific documentation is needed.

### Application Form Filling Protocol
1. **Register on ZED Portal:** Enter Udyam Registration Number and associated mobile number at https://zed.msme.gov.in/
2. **Validate Udyam:** System validates via API; ensure mobile number is accessible for OTP
3. **Provide Email ID:** Valid email required to receive ZED ID, password, and Pledge Certificate
4. **Take ZED Pledge:** Complete online pledge; Pledge Certificate issued immediately
5. **Login:** Use ZED ID and password to access dashboard
6. **Select Certification Level:** Choose Bronze, Silver, or Gold based on readiness and goals
7. **Download Mobile App:** Install ZED MSME app from Play Store (https://play.google.com/store/apps/details?id=com.qci.zed.msme) or App Store
8. **Provide GST/TAN:** Enter if applicable (mandatory field in app)
9. **Apply Joining Reward:** Opt-in for Rs. 10,000 reward (if within 1 year of pledge); amount deducted from certification fee
10. **Pay MSME Contribution:** Pay online via gateway after subsidy and reward adjustment (non-refundable)
11. **Upload Documents via App:** Submit selfie, site exterior photos, and basic information/documents
12. **Assessor Verification:** Await assessment; respond to non-conformities within stipulated time (30 days for Bronze, 60 days for Silver/Gold surveillance)
13. **Certificate Issuance:** Upon approval, certificate sent to email and available on dashboard
14. **Surveillance (if applicable):** For Silver/Gold, schedule Surveillance 1 (12–18 months) and Surveillance 2 (24–30 months)

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Submit application via ZED MSME mobile app after document upload
- Take screenshot of submission confirmation page showing timestamp and reference number
- Save reference number in case file
- Set calendar reminders for:
  - Follow-up at 7 days if no assessor allocation
  - Surveillance 1: 12–18 months post-certification (Silver/Gold only)
  - Surveillance 2: 24–30 months post-certification (Silver/Gold only)
  - Joining reward expiry: 1 year from ZED Pledge date

### Approval Actions
- [ ] Approval letter received via email and available on ZED dashboard
- [ ] Verify sanctioned amount: Certification fee minus joining reward (if applied) minus subsidy (based on enterprise type and eligibility)
- [ ] Utilization Certificate: Not required for certification subsidy; funds are disbursed as reduced payment at source
- [ ] Invoice success fee: [TO BE FILLED BY CONSULTANT] (as per firm’s internal policy)
- [ ] Monitor surveillance assessment due dates (Silver/Gold: Surveillance 1 at 12–18 months, Surveillance 2 at 24–30 months)
- [ ] Ensure handholding support application (if desired) is submitted after achieving at least Bronze certification
- [ ] Confirm free MSME KAWACH (WASH Certification) access post-ZED Pledge
- [ ] Explore international certification support (e.g., CE marking) via ZED scheme provisions
- [ ] Maintain records: Certificate, assessment report, payment receipts, and all correspondence for 3 years (validity period)
