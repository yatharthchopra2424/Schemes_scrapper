# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- ✅ The applicant is an MSME (Micro, Small, or Medium Enterprise) as defined under the MSMED Act, 2006  
- ✅ The applicant has a valid Udyam Registration Certificate (URC)  
- ✅ The applicant operates in any sector (manufacturing, service, or trading)  
- ✅ The applicant is located anywhere in India (Pan-India geographic scope)  
- ✅ The applicant seeks support in finance, marketing, technology, or capacity building  
- ✅ The applicant requires redressal, guidance, or handholding for MSME-related issues  
- ✅ The applicant wishes to access information in one of the eleven regional languages via State control rooms  

*Source: MSME Champions Scheme Key Facts; Application Process; Benefits; Objectives*

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Udyam Registration Certificate  
- PAN Card  
- Aadhaar Card  
- GST Certificate and Returns  
- Income Tax Returns (past 1–3 years)  
- Bank Statements (6–12 months)  
- Business address and ownership proof  
- Financial Statements (Balance Sheet, P&L, Cash Flow)  

### Scheme-Specific Documents
- No scheme-specific documents are required beyond the universal set, as the Champions Portal does not provide direct financial support but facilitates access to finance through guidance and handholding  

### Document Quality Check
- File size: Each document must be under 5 MB  
- File naming: Must match the applicant’s legal name as per Udyam Registration  
- Certification: Scanned copies must be legible and certified true copies where applicable  
- Language: Documents must be in Hindi or English; regional language support available via State control rooms  
- Validity: All documents must be current (not expired)

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
Not required. The Champions Portal does not require a Detailed Project Report (DPR) as it does not provide direct financial support but offers guidance and handholding.  
*Flag: Verify with nodal officer if any specific guidance document is needed for handholding services.*

### Application Form Filling Protocol
1. Access the Champions Portal via the Ministry’s website: https://msme.gov.in  
2. Navigate to the “Champions Portal” section (visible on the homepage footer and sitemap)  
3. Select your preferred language from the eleven available regional languages  
4. Clearly state the nature of your query or support needed (finance, marketing, technology, capacity building, redressal, etc.)  
5. Provide accurate Udyam Registration Number and business details  
6. Attach supporting documents if seeking specific scheme guidance (e.g., PMEGP, CGTMSE)  
7. Ensure all information matches your Udyam Registration details  
8. Submit the query and note the reference number for tracking  
9. Expect a response from the relevant State Control Room or nodal officer within the stipulated SLA  
10. Follow up using the escalation path if no response is received  

*Source: MSME Champions Scheme Key Facts; Application Process; Portal: https://msme.gov.in; Supporting Evidence: Sitemap, Citizen’s Charter, Logo Support pages*

### Internal Review Gate
- ✅ All fields in the query form are complete  
- ✅ Udyam Registration Number is valid and matches business details  
- ✅ Supporting documents (if any) are attached and legible  
- ✅ Language preference is selected  
- ✅ Query category (finance, marketing, etc.) is correctly identified  
- ✅ Contact details are accurate for follow-up  
- ✅ Submission screenshot and reference number are saved

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Submit the query through the Champions Portal at https://msme.gov.in  
- Take a screenshot of the submission confirmation page showing the reference number  
- Store the reference number securely in the case file  
- Set a follow-up reminder for 7 days after submission

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- ✅ Receive and save the guidance/response from the Champions Portal or State Control Room  
- ✅ Verify that the advice aligns with the requested support area (finance, marketing, technology, capacity building)  
- ✅ If the guidance leads to a specific scheme (e.g., PMEGP, CGTMSE), initiate application for that scheme using its respective playbook  
- ✅ Utilize the information in the client’s preferred regional language if applicable  
- ✅ Document the outcome and update the client on next steps  
- ✅ Utilization certificate not applicable (as no funds are disbursed directly via Champions Portal)
