# Application Playbook

## How to Use This File
For each new client case, assign it a Case ID, pick the target scheme, and follow the steps below.

---

## Stage 1: Eligibility Verification (Day 1–3)

### Checklist
- [ ] Applicant is an agri-startup, agri-entrepreneur, farmer, farmer producer organization (FPO), self-help group (SHG), cooperative, NGO, or MSME engaged in agriculture and allied sectors  
- [ ] Project idea is innovative, feasible, and has potential for scalability and impact on farmers' income  
- [ ] Project promotes value addition, post-harvest management, or market linkage (preference given)  
- [ ] Project does not involve pure trading activities or land purchase  
- [ ] Applicant is based in India (Pan-India geographic scope)  
- [ ] Applicant has not exceeded funding limits or violated scheme guidelines in prior applications (verify with nodal officer)  
- [ ] Project timeline is realistic and completion is feasible within stipulated period  

> **Sources**: Scheme Key Facts (Eligibility, Benefits, Key Caveats, Objectives); Application Portal: https://rkvy.nic.in/

---

## Stage 2: Document Collection (Day 3–10)

### Universal Documents (always required)
- Certificate of Incorporation / Registration (for startups, FPOs, NGOs, cooperatives, etc.)  
- PAN of the applicant entity  
- Aadhaar of authorized signatory  
- Bank account details (cancelled cheque or bank statement)  
- Udyam Registration (if MSME)  
- GST Registration Certificate (if applicable)  
- ITR of the entity for last 2 financial years (if applicable)  
- Audited financial statements (if applicable)  

### Scheme-Specific Documents
- Detailed Project Report (DPR) including technical, financial, and economic feasibility  
- Land lease/ownership documents (if project involves land use)  
- Quotations for equipment and machinery  
- Undertaking of compliance with scheme guidelines  

### Document Quality Check
- [ ] All documents are legible, scanned in PDF format (max file size: 5 MB per file)  
- [ ] Name on all documents matches exactly with the applicant entity name  
- [ ] Authorized signatory's Aadhaar and PAN are linked and valid  
- [ ] Bank account is in the name of the applicant entity  
- [ ] Undertaking is on stamp paper (if required by state) and duly notarized  
- [ ] DPR is signed and stamped by authorized signatory  
- [ ] Quotations are on vendor letterhead, valid for at least 30 days, and include GST details  
- [ ] All documents are in English or Hindi; if in regional language, attach certified translation

## Stage 3: Application Drafting (Day 10–18)

### Project Report / DPR (if required)
**Required**: Yes, a Detailed Project Report (DPR) is mandatory for RKVY-RAFTAAR applications.  

**Standard Sections to Include**:  
- Executive Summary  
- Project Background and Objectives  
- Technical Description (innovation, technology, processes)  
- Market Analysis and Linkage Strategy  
- Financial Projections (costs, revenue, profitability, break-even)  
- Implementation Timeline (milestones, Gantt chart)  
- Risk Assessment and Mitigation Plan  
- Sustainability and Scalability Plan  
- Annexures: Quotations, Land Documents, Team CVs, Undertaking  

> **Sources**: Scheme Key Facts (Required Documents, Application Process, Benefits)

### Application Form Filling Protocol
1. Access the application via the State Agriculture Department’s designated portal (link available at https://rkvy.nic.in/ under “State-wise Implementation”)  
2. Select “RKVY-RAFTAAR” as the scheme and enter the correct financial year  
3. Fill all fields in block letters; avoid abbreviations unless officially recognized  
4. Ensure project title clearly reflects innovation, value addition, or market linkage  
5. Enter financial details exactly as per the DPR (no rounding or estimation)  
6. Upload documents in the specified order and format (PDF, <5 MB/file)  
7. Double-check Aadhaar and PAN linkage before final submission  
8. Preview the entire application before final submit; no edits allowed after submission  

> **Sources**: Scheme Key Facts (Application Process, Application Portal); Standard government form-filling practices

### Internal Review Gate
- [ ] All mandatory fields in the application form are filled  
- [ ] DPR is complete, signed, and aligned with scheme objectives (value addition, innovation, market linkage)  
- [ ] All scheme-specific and universal documents are attached and valid  
- [ ] Undertaking of compliance is signed and notarized (if required)  
- [ ] Bank details match the entity name and are active  
- [ ] No mention of pure trading or land purchase in project description  
- [ ] Financials in form match DPR (within 5% tolerance for estimation)  
- [ ] Application is previewed and confirmed before final submission

## Stage 4: Submission & Follow-Up (Day 18–30)

### Submission Protocol
- Take a screenshot of the confirmation page showing Application Reference Number and timestamp  
- Store the reference number in the case file with label: “RKVY-RAFTAAR_Application_Ref_[CaseID]”  
- Save a PDF copy of the submitted application and all attachments  
- Set calendar reminders for follow-up at 15, 30, and 45 days post-submission  
- Monitor the portal for updates or requests for additional information

## Stage 5: Post-Approval (Day 30–90+)

### Approval Actions
- [ ] Obtain and verify the official approval letter from the State Level Sanctioning Committee (SLSC)  
- [ ] Confirm sanctioned grant amount and installment schedule as per approved DPR  
- [ ] Utilize funds strictly as per the approved project plan and scheme guidelines  
- [ ] Submit Utilization Certificate (UC) and Progress Report for each installment before requesting next tranche  
- [ ] Maintain audited accounts and records of all expenditures for scrutiny  
- [ ] Ensure project completion within the stipulated timeline  
- [ ] Submit final project report, UC, and impact assessment for closure  
- [ ] Coordinate with consultant for success fee invoicing upon fund utilization verification  

> **Sources**: Scheme Key Facts (Financial Support, Application Process, Key Caveats)
